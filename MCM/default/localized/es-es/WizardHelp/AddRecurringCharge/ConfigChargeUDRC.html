[ES]Unit Name
[ES]The name appearing next to the unit entry field in MetraCare. For instance, Number of Conferencing Seats.

[ES]Min Unit Value and Max Unit Value
[ES]The maximum and minimum amounts the user can enter in this field.

[ES]Value Type
[ES]The permitted value type for this field (either integer or decimal).

[ES]Rating
  • [ES]Tiered : The system rates all units based on the final range reached in the rate schedule.
  • [ES]Tapered : The system rates each unit based on the range the unit falls into in the rate schedule.
