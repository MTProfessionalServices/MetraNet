Nom unité
Dans MetraCare, nom figurant en regard de la zone d'entrée d'unité. Par exemple, Nombre de sièges pour conférence.

Valeur unitaire Min. et valeur unitaire Max.
Montants maximum et minimum que l'utilisateur peut saisir dans cette zone.

Type de valeur
Type de valeur autorisé pour cette zone (nombre entier ou nombre décimal).

Tarification
  • Par niveaux : le système applique une tarification à toutes les unités sur la base de la plage finale atteinte dans l'échelle tarifaire.
  • Fixe : le système applique une tarification à chaque unité sur la base de la plage applicable dans l'échelle de tarification.
