Unit Name
The name appearing next to the unit entry field in MetraCare. For instance, Number of Conferencing Seats.

Min Unit Value and Max Unit Value
The maximum and minimum amounts the user can enter in this field.

Value Type
The permitted value type for this field (either integer or decimal).

Rating
  • Tiered : The system rates all units based on the final range reached in the rate schedule.
  • Tapered : The system rates each unit based on the range the unit falls into in the rate schedule.
