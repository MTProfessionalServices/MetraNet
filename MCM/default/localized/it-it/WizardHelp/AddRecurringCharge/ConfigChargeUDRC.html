[IT]Unit Name
[IT]The name appearing next to the unit entry field in MetraCare. For instance, Number of Conferencing Seats.

[IT]Min Unit Value and Max Unit Value
[IT]The maximum and minimum amounts the user can enter in this field.

[IT]Value Type
[IT]The permitted value type for this field (either integer or decimal).

[IT]Rating
  • [IT]Tiered : The system rates all units based on the final range reached in the rate schedule.
  • [IT]Tapered : The system rates each unit based on the range the unit falls into in the rate schedule.
