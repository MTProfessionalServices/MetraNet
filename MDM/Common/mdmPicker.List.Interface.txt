INTERFACE ?

The interface is the mandatory or optional parameters ([]), supply in the query string

Picker Parameters
_______________________________________________________________________________________________________________________________________

QueryString Input Parameters

    NextPage                    = URL where the page will go when the user select one item in the list.
    [Parameters]                = URL Parameters. The format is different from a query string for obvious reason.
                                  Syntax : name|value[;]
                                  Sample : PT_ID|2389;PI_ID|8799
    Title                       = A dictionary entry for the title
    [LinkColumnMode]            = List dialog support 2 mode
                                  Edit/View Icon Mode - default mode
                                  Link mode (no icon) - The query string must contains LinkColumnMode=TRUE

    [Kind]                      = Support to filter to kind of type.
    [MonoSelect]                = Multi or Mono Select picker. Default value MonoSelect=FALSE    

    [OptionalColumn]		= You can ask the picker to return you another column information for example the column n_name


QueryString OutPut Parameters

	When the user click OK the picker call the NextPage with as QueryString parameters
		IDS : Contains the list of item id selected in a CSV format
		OptionalValues : If OptionalColumn is defined, OptionalValues contains the list of column values asked in a CSV format

	If Parameters is defined the Parameters name=value is added to the query string.

List Parameters
_______________________________________________________________________________________________________________________________________

QueryString Input Parameters

    NextPage                    = URL where the page will go when the user select one item in the list.
    [Parameters]                = URL Parameter. The format is different from a query string for obvious reason.
                                  Syntax : name|value[;]
                                  Sample : PT_ID|2389;PI_ID|8799
    Title                       = A dictionary entry for the title
    [LinkColumnMode]            = List dialog support 2 mode
                                  Edit/View Icon Mode - default mode contains LinkColumnMode=FALSE
                                  Link mode (no icon) - The query string must contains LinkColumnMode=TRUE
    


QueryString OutPut Parameters

	When the user click on the View icon or, edit icon or link the list dialog call the NextPage with as QueryString parameters
		IDS : Contains the ID selected

	If Parameters is defined the Parameters name=value is added to the query string.
