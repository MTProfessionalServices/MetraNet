Metering with FLS

Problem

The File Landing Service provides an efficient means to consume data files generated from external or internal sources. However, the acquired data must often be processed by MetraNet's pipeline architecture and therefore needs to be "metered" - that is, matched up with a pipeline's service definition.

Solution
