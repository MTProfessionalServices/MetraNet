Verify or Start the Data Export Service

The Data Export Service must be running on the Billing Server you have selected for generating data export reports.

Do the following on the billing server or in a remote desktop session to the billing server.

  1. From the Start menu's Search programs and files, type Services.
  2. In Component Services, select the Services (Local) node.
  3. Verify that the status of MetraTech Data Export Service is Started.
  4. If it is not started, right-click MetraTech Data Export Service and select Start.
