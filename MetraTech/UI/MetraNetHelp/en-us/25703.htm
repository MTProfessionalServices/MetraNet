MetraOffer Advanced Options Manage AMP Decision Types

In the MetraOffer module, in the main menu, expand Advanced Options and click Manage AMP Decision Types.

The main page of the wizard appears. The Decision Types section lists all existing decision types, including preconfigured decision types that are provided with the wizard.

You can sort the columns by clicking the headings, or use the Search Filters to show fewer decision types.

Select a decision type from the list, scroll to the right, and click an action.
