Decision Cycle

MetraOffer Advanced Options Manage AMP Decision Types <selected decision type>

The decision cycle defines the time period during which the decision type aggregates (such as days, weeks, months) and how long the decision remains in effect.

  1. Provide the following information.

    Field

    Description

    Unit of time for the Decision Cycle
    • Decision matches the billing interval—The billing interval is set in the MetraCare module, in the "Billing Cycle" property for the user's account.

    How long should the Decision Type be in effect?
    • Indefinitely—The decision cycle repeats as long as the decision type is active.
    • For a specific number of Decision Cycles—The decision is processed for the number of times you specify. Provide a fixed value or a parameter table column. If the decision type is made inactive, any decision cycles that are partially complete will stop processing.
  2. Click Save & Continue.
  3. Continue with the Decision Action wizard page.
