About Usage Qualifications

Usage—A rated billing record that is a collection of product view record fields.

Usage Qualification Group—A set of rules that determine which usage events should be considered by a decision.

When the AMP engine runs, it evaluates all the product view records that belong to an account for a given time span. Usage records whose properties satisfy the rules are analyzed by the decision type. Usage records that do not satisfy the rules are ignored by the decision type.

Any usage qualification can be used by any decision types.

Select Usage Qualification page

The page lists all existing usage qualifications. You can view their properties but not edit them.

  1. Select a usage qualification for this decision type. Click Save & Continue.
  2. Continue with the Items to Aggregate page.
