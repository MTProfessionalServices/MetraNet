MetraControl Tasks

The MetraControl module lets you maintain and monitor MetraNet information through a graphical user interface. Operations staff use the MetraControl module for:

  • Billing Management
    Monitoring and managing billing intervals and billing groups.
  • Scheduled Adapters
    Monitoring and running scheduled business process adapter processing.
  • Usage Server Maintenance
    Running usage server maintenance tasks.
  • Batch Management
    Monitoring and managing batch transactions.
  • Backout Management
    Monitoring and managing the rollback and rerun of transactions.
  • File Management
    Managing incoming data files on a MetraNet server.
  • Metering Statistics
    Monitoring data about processed transactions.
  • Failed Transactions
    Monitoring and managing transactions that could not be processed successfully.
  • Administration
    Viewing file versions, editing namespaces, specifying machine roles, managing reports, and synchronizing with the database.
