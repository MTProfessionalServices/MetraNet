This page shows the roles and capabilities for the current account.

Roles

From this page, you can work with roles:

  • Click Add Role to give a system user a new role.

If a role has been added, you can:

  • Click View to see the details of a role.
  • Click Delete to remove a role.

Capabilities

From this page, you can work with capabilities:

  • Click Add Capability to add a capability.

If a capability has been added, you can:

  • Click Edit to set a capability's details (such as for the Application Logon capability).
  • Click Delete to remove a capability.
