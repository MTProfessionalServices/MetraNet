Account State

Account Menu Bar Account Details Account States

This page displays a history of account states for the current account, with the start and end date for each state.

An account's state determines the actions you can perform on it.

To edit the account state effective date:

  • Click Edit to change the effective date for the current account state.

To update an account state:

  • Click Update Account State to change the account state.
