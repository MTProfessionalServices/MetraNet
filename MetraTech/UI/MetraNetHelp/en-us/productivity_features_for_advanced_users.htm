Productivity Features for Advanced Users

MetraOffer adds a host of improvements to the user interface that increase your everyday productivity within MetraOffer. Among these are:

  • Create a copy of a price list with a different currency.
  • Do bulk updates of price list mappings and ICB information.
  • See summary information in product offering parameter tables.
  • Tabbed editing of calendars.
  • One-click creation of a default rate schedule.
  • Clone an existing rate schedule to create a new one.
  • Easily edit the custom recurring charge edit screen.
  • View MetraOffer pages full screen if desired.
