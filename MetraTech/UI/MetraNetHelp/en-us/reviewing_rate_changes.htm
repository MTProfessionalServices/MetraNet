MetraOffer maintains a detailed history of each rate update for review and auditing purposes.

When you edit a rate schedule, you can review the rule changes you made using a color-coded "diff" feature. Click View Changes to open the Ruleset Changes window.

The Ruleset Changes window shows Added and Deleted when a row is moved up or down, as well as when a row is added or deleted.
