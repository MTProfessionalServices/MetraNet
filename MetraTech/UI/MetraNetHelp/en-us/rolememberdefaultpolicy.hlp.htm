View Default Security Administrators

Clicking the View Default Security Administrators on the Role Members page opens this page. This is useful information when you want to remove a role, and need to understand where it is being applied before removing it.

This page lists all accounts that have this role applied on their default security policy. This includes internal system folders. Internal system folders hold the default security policies for different types of accounts, such as Independent, MetraControl, MetraOffer, and CSR accounts.

Changing the default security policy on these folders must be performed by your system integrator.
