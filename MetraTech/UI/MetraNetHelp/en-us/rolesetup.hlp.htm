View or Edit Capabilities for Role

This screen lists the details of capabilities associated with the selected role.

You can access this page from either the Default Security or Manage Roles command.

View Capabilities

To view the capabilities associated with a default role:

  1. From the Security menu, select Default Security.
  2. Click View .

Edit Capabilities

To view or edit any role in the system:

  1. Select Manage Roles from the Security menu.
  2. Click Capabilities for the capability you would like to edit.

From this page, you can configure capabilities for the selected role in any of the following ways:

  • Add a Capability to the role by clicking Add Capability.
  • Edit a capability by clicking the Edit next to the capability you wish to change.
  • Delete a capability from a role by clicking the Delete next to the capability you would like to remove.
