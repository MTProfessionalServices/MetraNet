Scheduled Adapters Event List

This page lists all the scheduled adapters presently scheduled to run in the system.

  • To view the details of any adapter (last runs, status of each run, and so on.) click the adapter name.
  • To specify the schedule for running the adapter, click the Every <recurrence pattern> or Next Run links.

    Field

    Description

    Adapter

    The full name of the adapter.

    Schedule

    The configurable schedule (or recurrence pattern) with which the billing server runs this adapter. There are two options you can click.

    Every <recurrence pattern>

    Shows the adapter run schedule. Click to open the Recurrence Pattern page to change the schedule.

    Next Run

    Shows the next scheduled adapter run. Click to open the Override Schedule page to change the schedule once.

    Description

    A description of what the adapter does.

In the search bar, you can sort or filter this list by Adapter, Schedule, or Description. For more information, see About the Search Function.
