Using MetraControl Search Functions

Many MetraControl windows have a toolbar for paging or searching through records.

To search or filter a list of records in MetraControl:

  1. In the drop-down list, choose the field to search (the drop-down list is different for each set of records you search).

  2. Choose the operator you want to use in the search.

    Operator

    Description

    =*

    The search will return all values containing the value you enter.

    =

    The search will return all values exactly corresponding to the value you enter.

    >

    The search will return all values alphanumerically greater than the value you entered.

    >=

    The search will return all values alphanumerically equal to or greater than the value you entered.

    <

    The search will return all values alphanumerically less than the value you entered.

    =<

    The search will return all values alphanumerically less than or equal to the value you entered.

    <>

    The search will return all values not equal to the value you entered.
  3. Enter the value you want to search for.
  4. Click Go.

    If you click Reset the search criteria clears and the records appear in their default sort order (as they appeared when first entering the screen).
