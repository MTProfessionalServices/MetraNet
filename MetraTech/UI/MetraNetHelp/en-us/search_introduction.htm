Find an Account

You can find and select accounts quickly using the Find an Account field.

  1. Type one, two, or more characters.

    As you type, Find an Account auto-suggests account names. You can see immediately the account names that begin with these characters.

    • You can use partial words and wildcards.
    • If many names are returned, you can scroll through the results list, page by page.
    • If you need to see more names and details, click Advanced Find (or use Account Finder).
  2. Click the account you want to make your active account.

    The account details open and the Account menu bar appears.
