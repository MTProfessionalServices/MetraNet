MetraNet Buttons and Icons

Button

Button Name

Action

Edit

Lets you edit information, such as transaction details.

View Details

Expands a line item in a list so you can view more details.

View

Lets you view detailed information.

Remove

Removes an entry.

OK

Confirms the acceptance of an entry you make in the Edit dialog box.

Cancel

Cancels your entry and returns you to the previous page.

Close

Closes the present window and returns you to the previous one.

Add

Activates an Add page where you can add information.

Refresh

Updates the page display and data to show recently-made changes.

First Page

Returns you to the first page of a list.

Previous Page

Returns you to the previous page of a list.

Next Page

Moves you to the next page of a list.

Last Page

Moves you to the last page of a list.
