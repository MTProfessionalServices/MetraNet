Anzeigen des Zahlungsverlaufs

Auf der Seite Zahlungsverlauf werden alle zurückliegenden Zahlungen aufgeführt, die Sie mit diesem Konto getätigt haben.

So zeigen Sie den Zahlungsverlauf an:

  • Wählen Sie in MetraView Rechnung und Zahlungen » Zahlungsverlauf. Daraufhin wird eine Liste aller kürzlich vorgenommenen Zahlungen angezeigt.

Mithilfe der Suchfilter können Sie die auf der Seite angezeigten Daten eingrenzen, eine gespeicherte Suche anlegen, um diese Daten ganz einfach erneut anzeigen zu können, und die Daten in eine CSV-Datei exportieren. Weitere Informationen hierzu finden Sie unter Filtern, Suchen und Sortieren von Daten in einer Online-Rechnung und Exportieren von Daten in eine CSV-Datei.
