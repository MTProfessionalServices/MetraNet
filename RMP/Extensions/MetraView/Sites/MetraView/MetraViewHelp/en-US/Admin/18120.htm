Editing the Dashboard

The Admin Site's Dashboard contains a collection of Columns, layout components (which are themselves the containers for widgets and parameters. The Dashboard is therefore a container for the objects and properties defined by these other entities. The Dashboard, itself, is contained by the Site.

All the Dashboard components are BMEs. This means that all changes you make to these components are stored in the database. In order to move BME data from one development system to another or to a production system, you can use the BMEImportExport tool.

To access the Dashboard-related components:

  1. From Configure Site, select the instance of the site you want to work with and click Edit.
  2. Under More Items, click the Dashboard component.

    Do not use View Relationships to select the Dashboard.

To edit the Dashboard properties:

  • For the Dashboard instance you want to work with, click Edit to open the Edit Dashboard page.

To add, view, edit, or delete a Column:

  1. For the Dashboard instance you want to work with, click View Relationships.
  2. Click Add and define the Column properties.

To add, view, edit, or delete a Widget:

  1. For the Column instance you want to work with, click View Relationships.
  2. Click Add and define the Widget properties.

To add, view, edit, or delete a Parameter:

  1. For the Widget instance you want to work with, click View Relationships.
  2. Click Add and define the Parameter properties.
