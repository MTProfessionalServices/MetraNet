Configuring a Site

The Configure Site menu item is your single point of access to all the configurable components of a MetraView site's user interface (UI). Starting here you can edit the site properties and you can add, edit, or remove any site component.

To edit the site properties:

  • From the Site list, for the site you want to work with, click Edit to open the Edit Site page.

    A Site is a nested collection of user interface controls contained in an instance of the MetraView web framework. When you drill down through a site you are selecting the control type and then one instance of that control.

    See how to navigate the site components.

    User interface controls or components are called Business Modeling Entities (BMEs) elsewhere in MetraNet documentation for system integrators and configuration developers. For clarity, this Help refers to them simply as components.

Site components

Component

Purpose

BillSetting

Sets MetraView online bill properties.

Dashboard

Contains the Column and Widget components.

EntryPoints

Sets a nonstandard entry point to the site.

ProductViewMappings

Sets the display of usage information for a particular service.

ReportInventories

Manages the subscriber report types.
