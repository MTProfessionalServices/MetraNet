Editing the Site Components

The Site is a container for the various components that comprise a MetraView site's user interface. From the Site instance, you can create, edit, and delete instances of these components. You can access all the properties of these individual components through the Site container.

All component changes should be thoroughly tested before going live with the site.

Navigating the Site Components

Beginning with the site instance, all components are related to their container component in a one-to-many or many-to-many relationship. For example, you can define one or more custom Dashboard instances.

There are two ways to navigate site components: using relationships and using simple drill-down. Which one you use depends on where you are in the component hierarchy.

To add, view, edit, or delete the top-level site components:

  • From the Site list, for the site instance you want to work with, click View Relationships to open the Relationships for <YourSite> page.

To add, view, edit, or delete subcomponents:

  1. From the component list, expand () the site instance you want to work with, so that you see the component icons under More Items.
  2. Click the component type you want to work with.

    When you do this, the subcomponent list, such as for Dashboards, will show the View Relationships option.
