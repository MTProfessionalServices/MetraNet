Entry Points

The Entry Points component sets up a nonstandard entry point to the site.

Entry points use a secure "ticketing" method that permits users authenticated by a trusted site to enter particular pages in your MetraView site directly, without having to go through a login sequence. Users are authenticated using a ticket that is generated by a MetraNet-supplied ASPX file after their logon name/password is validated.

Parameter

Description

Entry Point Name

A name that uniquely identifies the entry point.

Description

A description of the purpose of the entry point and the users who can access it.

URL

A entry-point page URL.
