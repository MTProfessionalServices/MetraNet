Columns

The Columns component determines how many columns the Dashboard contains and the position and content of each column.

To drill down to columns:

  • Select Site » Dashboard » Columns

    Parameter

    Description

    Name

    A unique name for the column

    Description

    An optional description of the column

    Title

    The title of the column that may appear at the top

    Position

    The position of the column in the dashboard (1, 2, 3)

    Width

    Width of the column as a decimal fraction of the dashboard. (For example an even-sized 3 column dashboard would have 3 columns with widths of .33)

    CssClass

    An override to CSS class used for the Site or Dashboard theme.

    Style

    An additional set of styles to be added to the rendered column.
