Site Properties

The top-level Site component is where you define basic properties of your site instance.

Parameter

Description

Site Name

A unique text string for identifying this site.

Description

A description of the site.

Is Site Down for Maintenance?

Choose this option whenever you make changes through the Configure Site interface. You would usually require a backup MetraView server during this time.

Culture

This is the localization setting for the site. The default is US English (en-US). Do not change the culture unless the corresponding localized files exist for the MetraNet platform.

Themes

Theme names and options are determined by the CSS style sheet. Do not change the theme unless the correspondingly named style sheet exists in RMP\Extensions\MetraView\Sites\<sitename>\Styles.

Logo Image

Specify the name of a logo image: GIF, JPG, or PNG. The file should be added to RMP\Extensions\MetraView\Sites\<sitename>\Images.
