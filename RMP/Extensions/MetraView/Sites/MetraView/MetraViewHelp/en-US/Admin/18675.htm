Parameters

For every widget, you have the option of setting optional parameters.

To drill down to widget parameters:

  • Select Site » Dashboard » Column » Widgets » Parameters

    Parameter

    Description

    Name

    The parameter name.

    Value

    The parameter value.

    Description

    A description of the purpose of the parameter

    Is Required

    Whether or not the parameter is required.
