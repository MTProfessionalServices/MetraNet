Accessing the Admin Site

To see, access, and modify a MetraView administrative site, you require the following special account:

  • MetraViewAdmin

    or

  • A subscriber account that has been given the following capabilities:
    • MetraView Admin, Read Business Modeling Entities, Write Business Modeling Entities

    For more information, see MetraView User Capabilities

    The MetraView URL can be http://<servername>/MetraView or http://<servername>/<SiteName> or https://<servername>/MetraView or https://<servername>/<SiteName>

The Admin tab is visible when you log into a MetraView site with one of these roles.

The Admin site provides access to all the configurable components of a MetraView site. Starting here you can create new site instances, and you can add, edit, or remove any predefined site component.

You can add shared reports, which are query-driven grids, and new widgets, which are standard ASCX controls. To create new Business Modeling Entities (BMEs), you require MetraNet ICE. For more information, search the Documentation Set.
