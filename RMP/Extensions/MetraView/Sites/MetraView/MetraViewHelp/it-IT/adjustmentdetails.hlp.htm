Visualizzazione dei dettagli delle rettifiche

Se sono state apportate rettifiche al conto, queste informazioni sono visualizzate sulla pagina Dettagli rettifica.

Per visualizzare le rettifiche:

  1. In MetraView, selezionare Fatturazione e pagamenti.
  2. Nella sezione Pagamenti, crediti e rettifiche, selezionare la rettifica che si desidera visualizzare. Verrà visualizzata la pagina Dettagli rettifica.

    È possibile utilizzare i filtri di ricerca per limitare i dati visualizzati nella pagina, creare una ricerca salvata per ritrovare rapidamente i dati in seguito ed esportare i dati all'interno di un file CSV. Per ulteriori informazioni, vedere le sezioni Filtraggio, ricerca e ordinamento dei dati su una fattura in linea ed Esportazione di dati in un file CSV.

  3. Fare clic su Indietro per tornare alla pagina Riepilogo fatture.
