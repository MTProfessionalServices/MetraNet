Accessing the Admin Site

To see and access a MetraView administrative site, you require one of the following special capabilities:

  • Admin
  • MetraViewAdmin

The Admin tab is visible when you log into a MetraView site with one of these capabilities.

The Admin site provides access to all the configurable components of a MetraView site. Starting here you can create new site instances, and you can add, edit, or remove any predefined site component. You can also add new widgets, which are standard ASCX controls. However, you cannot create new Business Modeling Entities (BMEs). For this you require MetraNet ICE.
