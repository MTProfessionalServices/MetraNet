Exibir detalhes de ajuste

Se sua conta tiver sofridos ajustes feitos por isso, você pode exibir estes detalhes na página Detalhes de ajuste.

Para exibir ajustes:

  1. Em MetraView, selecione Fatura e pagamentos.
  2. Em Pagamentos, créditos e ajustes, selecione o ajuste que deseja ver. A página Detalhes de ajuste é exibida.

    É possível usar os filtros de busca para restringir os dados que você vê nesta página, criar uma busca salva para encontrar facilmente esses dados novamente e exportar dados para um arquivo CSV. Para obter mais informações, consulte Filtrar, buscar e classificar dados em uma fatura online e Exportar dados para um arquivo CSV.

  3. Clique em Voltar para retornar à página Resumo de fatura.
