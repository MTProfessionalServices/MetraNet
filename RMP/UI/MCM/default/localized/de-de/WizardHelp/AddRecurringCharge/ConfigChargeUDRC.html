[DE]Unit Name
[DE]The name appearing next to the unit entry field in MetraCare. For instance, Number of Conferencing Seats.

[DE]Min Unit Value and Max Unit Value
[DE]The maximum and minimum amounts the user can enter in this field.

[DE]Value Type
[DE]The permitted value type for this field (either integer or decimal).

[DE]Rating
  • [DE]Tiered : The system rates all units based on the final range reached in the rate schedule.
  • [DE]Tapered : The system rates each unit based on the range the unit falls into in the rate schedule.
