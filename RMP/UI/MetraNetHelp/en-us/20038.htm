Add Parameter Table to Price List

The Add Parameter Table to Price List window lets you select a priceable item or existing parameter table and create a rate schedule for it.

  1. From Create Rate Schedules, select a usage charge (which must have been previously defined for you), a recurring charge (RC), a nonrecurring charge (NRC), or a discount.
  2. Enter an optional description for the rate schedule.
  3. Enter an effective start date and end date (or do not specify).
  4. Click Create Rate Schedule(s).
  5. In the Selected Price List box (as illustrated), click the newly created rate schedule to edit its properties, including defining a parameter table (where applicable).
