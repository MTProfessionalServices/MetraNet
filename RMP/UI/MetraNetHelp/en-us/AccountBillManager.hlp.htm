Bill Manager

Account Menu BarRelationships Bill Manager

This page lets you search for, add, and delete bill managers for the active account. A bill manager is a special category of subscriber account that is responsible for the bills of other subscribers. When bill managers log into MetraView, they see a list of the subscriber accounts they are responsible for.

For this relationship to exist between subscribers:

  • A subscriber must be designated an account bill manager using the Bill Manager command.
  • The bill manager must have the Manage Account Hierarchies with Write access and Impersonation capabilities. (See Setting up the Manage Account Hierarchies Capability)

To add a bill manager for the active account:

  1. Click Add to open the Select Account dialog.
  2. Select a subscriber account to be the bill manager the active account

    You can select more than one bill manager for the active account.
