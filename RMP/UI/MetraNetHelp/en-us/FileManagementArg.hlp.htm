You configure one argument at a time:

  • Enter argument Format, (optional) Order, Filter, and Conditional fields.

    If you specify the order - using 1, 2, and so on - the arguments will appear in this order on the MetraControl page.

    For MetraFlow scripts, the order of arguments does not matter. However, the argument names must always match the script parameters.

    When you specify an argument, there can be no spaces except for one after the arg keyword.

See Target Arguments for instructions about what to enter in these fields.
