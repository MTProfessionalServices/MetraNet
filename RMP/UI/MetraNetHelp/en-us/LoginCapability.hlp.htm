Using the Application Logon Capability

From this screen, you can determine which application(s) to apply the selected Application Logon capability. Depending on how your capabilities are configured, users who have it assigned to them can log into the applications that are listed.

The application logon capability is a necessary but not usually sufficient condition to enable use of a web application.

For System Users (CSRs):

Valid choices are any combination of one or more of the following: MetraOffer, MetraCare, MetraControl, MetraView.

For Subscribers

The only valid choice is MetraView.

Subscribers Cannot Log Into the MetraOffer, MetraControl, and MetraCare modules

The MetraOffer and MetraControl, and MetraCare modules are accessible to system users only. In the MetraCare module, system accounts are referred to as CSR accounts. (Even though the Application Logon capability is available for subscriber accounts in the MetraCare and MetraOffer applications, the system does not allow subscriber accounts to log into these applications.)
