Miscellaneous Attributes

MetraOffer Advanced Options Manage AMP Decision Types <selected decision type>

This feature requires advanced knowledge of AMP processing.

To configure a decision type to perform RAMP decision processing

  1. In the Miscellaneous Attributes area, click Add.
  2. Provide the following information.

    Field

    Provide this value

    Name

    tier_responsiveness

    Value dropdown list

    Fixed Value

    Field to specify the value

    realtime

  3. Click OK.
  4. Continue with the Error Check page.

To configure a decision type to calculate early termination charges

  1. Create a Commitment decision.
  2. In the Miscellaneous Attributes area, click Add.
  3. Provide the following information.

    Field

    Provide this value

    Name

    allow_forced_finalization

    Value dropdown list

    Fixed Value

    Field to specify the value

    1
