Adjust Transactions

You can adjust prebill or postbill transactions. When making individual adjustments, individual transactions are referred to as parent transactions. When making adjustments to parent-child (multipoint) transactions.

To adjust transactions:

  1. Select the adjustment type and enter the adjustment amount or percentage. Provide a reason code and description.
  2. Compute the adjustment to validate it and to see the final amount.
  3. Click Save when satisfied.
  4. Click the Save button to create this adjustment for the parent transaction. Otherwise, click Cancel.
  5. MetraCare displays an Operation Information dialog with an appropriate adjustment confirmation message or an appropriate error message.

    Depending on your MetraCare capabilities, your confirmed adjustments will either be automatically approved or updated with a pending status. CSRs with appropriate capabilities will later review, approve, or delete pending adjustments using the System menu's Manage Adjustment option.
  6. The Found Transactions page displays again showing the various icons changed to reflect your adjustment, such as the sample shown here . 

    Notice that the  Reassign icon is now grayed out (because you can no longer reassign this adjusted transaction to a different Account), and the former Adjust Transaction icon has been replaced with the  Manage Parent Adjustment icon.  

    Notice also that you can still click on the   Adjust child transaction icon to view and create adjustments for one or all of the child transactions associated with this adjusted Parent Transaction.
