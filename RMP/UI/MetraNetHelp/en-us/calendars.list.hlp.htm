Manage Calendars

This page lists all calendars presently in the system. Configuration developers add calendar parameter tables to usage charges in the MetraNet ICE tool.

  • To edit a calendar, click Edit .
  • To view a calendar, click View .

This page does not provide information on which priceable items and product offerings each calendar is associated with. For that information, you will need to view a priceable item's associated rate schedules.
