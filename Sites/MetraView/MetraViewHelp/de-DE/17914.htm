Exportieren von Daten in eine CSV-Datei

So exportieren Sie Daten:

  1. Filtern Sie in MetraView oder MetraCare die Daten, die Sie exportieren möchten.
  2. Wählen Sie unten in der Tabelle die Option Exportieren.
  3. Wählen Sie eine der folgenden Optionen aus:
    • Nur aktuelle Seite exportieren
    • Alle Datensätze exportieren
  4. Wählen Sie Exportieren.

    Wenn Sie einen Satz von Daten exportieren, der untergeordnete Daten enthält, müssen Sie angeben, welche der untergeordneten Daten in den Export einbezogen werden sollen. Beim Exportieren nach Excel werden die untergeordneten Daten in separate Registerkarten exportiert. Beim Exportieren in eine einfache Datei wird für jeden untergeordneten Datensatz eine separate Datei angelegt.

  5. Wählen Sie nun aus, ob Sie die erzeugte Datei öffnen oder speichern möchten. Die Daten werden im CSV-Format gespeichert.
