Anzeigen von Zahlungen, Guthaben und Anpassungen

Wenn Sie einzelne Nutzungsanmeldungen und Anpassungen anzeigen, werden die Details in einer Tabelle angezeigt, die sortiert, in mehrere Seiten aufgeteilt und exportiert werden kann. Mithilfe der Suchfilter können Sie die auf der Seite angezeigten Daten eingrenzen, eine gespeicherte Suche anlegen, um diese Daten ganz einfach erneut anzeigen zu können, und die Daten in eine CSV-Datei exportieren. Weitere Informationen hierzu finden Sie unter Filtern, Suchen und Sortieren von Daten in einer Online-Rechnung und Exportieren von Daten in eine CSV-Datei.

So zeigen Sie eine Rechnung aus einem anderen Rechnungszeitraum an:

  • Klicken Sie im Abschnitt Intervall auswählen auf das Dropdown-Feld, und wählen Sie das gewünschte Intervall aus.

Info zur Übersichtsansicht Ihrer Zahlungen, Guthaben und Anpassungen

In der Übersichtsansicht werden die Gesamtnutzungskosten pro Nutzungsanmeldung angezeigt, die von Ihrer Anmelde-ID getragen werden. Wählen Sie eine Nutzungsanmeldung aus, um die Kosten und Nutzungsdetails für diese Nutzungsanmeldung anzuzeigen. Wenn Sie die Nutzung nach Benutzern in Ihrem Konto aufschlüsseln möchten, wählen Sie die Detailansicht.

Info zur Detailansicht Ihrer Zahlungen, Guthaben und Anpassungen

In der Detailansicht werden die Gesamtnutzungskosten pro Nutzungsanmeldung für einen bestimmten Benutzer angezeigt. Wählen Sie die Nutzungsanmeldung aus, um die Kosten und Nutzungsdetails für einen Benutzer in Ihrer Hierarchie anzuzeigen. Die Gebührendetails sind nach den einzelnen Benutzern untergliedert, die dem Konto zugewiesen sind. Wählen Sie den gewünschten Benutzer aus, um eine Liste seiner Nutzungsanmeldungen anzuzeigen. Wählen Sie die Nutzungsanmeldung aus, um die Kosten-, Nutzungs- und Anpassungsdetails für diesen Benutzer anzuzeigen.
