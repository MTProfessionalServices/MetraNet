Filtern, Suchen und Sortieren von Daten in einer Online-Rechnung

Sie können die Darstellung der Daten ändern, die angezeigt werden, wenn Sie in MetraView die Nutzung, den Zahlungsverlauf oder die Anpassungen aufrufen. Wenn die Daten in Form einer Tabelle angezeigt werden, können Sie im Fenster "Filter konfigurieren" die Eigenschaften festlegen, die in der Suche und im Filter auf dieser Seite angezeigt werden.

So konfigurieren Sie die Suche und die Filter:

  1. Wählen Sie in der rechten oberen Ecke des Such- bzw. Filterbereichs der Tabelle das Symbol Filter konfigurieren.

    Die Konfigurationsoptionen werden angezeigt.

  2. Wählen Sie die Elemente aus, die in den Filter einbezogen werden sollen. Wenn eine Eigenschaft nicht abgewählt werden kann, wurde sie so konfiguriert, dass sie in der Suche bzw. im Filter immer angezeigt wird.

  3. Klicken Sie auf OK.

    Nach dem Filtern, Suchen und Sortieren können Sie diese Datenansicht zur Verwendung mit anderen Rechnungsdaten und Perioden speichern. Speichern einer benutzerdefinierten Suche in MetraView
