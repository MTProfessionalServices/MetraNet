Hinzufügen einer Kreditkarte

So legen Sie eine Kreditkarte als Zahlungsart für Ihr Konto fest:

  1. Wählen Sie Rechnung und Zahlungen » Zahlungsarten.
  2. Klicken Sie auf Kreditkarte hinzufügen.
  3. Geben Sie die Angaben zu Ihrer Person ein.
  4. Wählen Sie eine Priorität. Mit der Priorität wird die Position festgelegt, an der die Zahlungsart in der Liste der Zahlungsarten aufgeführt wird. Wenn dies Ihre erste Zahlungsart ist, verwenden Sie 1. Wenn Sie bereits eine Zahlungsart definiert haben, ändern Sie diese Zahl entsprechend der Position, an der die Zahlungsart in der Liste aufgeführt werden soll. Geben Sie 2 ein, wenn sie an zweiter Stelle aufgeführt werden soll usw.
  5. Geben Sie die Angaben zur Kreditkarte ein:
    • Wählen Sie den Kreditkartentyp aus.
    • Geben Sie die Kartennummer ein.
    • Geben Sie die Kartenprüfnummer ein.
    • Geben Sie das Gültigkeitsdatum der Karte ein.
  6. Klicken Sie auf OK. Die Zahlungsart wird zu Ihrem Konto hinzugefügt.

    Sie können mehrere Kreditkarten zu Ihrem Konto hinzufügen und beim Bezahlen der Rechnung jeweils die gewünschte Kreditkarte auswählen.
