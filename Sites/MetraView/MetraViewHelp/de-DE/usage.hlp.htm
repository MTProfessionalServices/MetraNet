Meine Berichte

Auf der Seite Meine Berichte werden eine Liste der Produktangebote, für die Sie angemeldet sind, und Ihre Nutzungsgebühren für einen bestimmten Zeitraum angezeigt. Wenn Ihre Rechnungen für ein Unternehmen ausgestellt werden, wird eventuell auch die Nutzung für Einzelpersonen in der Gruppe angezeigt, für die Sie verantwortlich sind. Dies wird als Hierarchieansicht bezeichnet.

Sie können zwischen einer Übersichtsansicht und einer Detailansicht für Ihre Nutzungsanmeldungen wechseln.

  • Die Übersichtsansicht zeigt die Gesamtnutzung für die einzelnen Leistungen, für die Sie angemeldet sind.
  • Die Detailansicht zeigt die Nutzung aller anderen Konten in Ihrer Hierarchie, für die Sie der Zahler sind.

So zeigen Sie Details zu Ihrer Nutzung an:

  1. Wählen Sie in MetraView die Registerkarte Meine Berichte. Daraufhin wird der aktuelle Nutzungsbericht angezeigt. Im Widget "Nutzungsbericht" in der linken Spalte wird eine Liste weiterer Nutzungsberichte angezeigt.
  2. Wenn Sie einen anderen Zeitraum anzeigen möchten, wählen Sie unter Verwendung anzeigen ab ein anderes Anfangs- und Enddatum aus, und klicken Sie dann auf Los.

    Nun wird eine Übersicht über die Nutzung innerhalb des ausgewählten Zeitraums angezeigt.

  3. Klicken Sie auf Detailansicht oder Übersichtsansicht, um eine Detailansicht bzw. eine Übersichtsansicht der Gebühren aufzurufen.

    Wenn Sie andere Teilnehmer verwalten, beispielsweise in einer Abteilung oder einer anderen Kostenstelle, werden in der Detailansicht die Gebühren für die einzelnen zahlenden Konten angezeigt.
  4. Wählen Sie bestimmte Gebühren für die einzelnen Berechnungen in einem Produktangebot aus. Sie können noch genauere Details zu diesen Gebühren anzeigen, indem Sie links neben der jeweiligen Gebühr auf das Erweiterungszeichen klicken.
  5. Wählen Sie die Registerkarte Meine Berichte, um zum Nutzungsbericht zurückzukehren.

    Wenn Sie die Nutzungsdetails anzeigen, können Sie die Ergebnisse filtern. Der hierzu erstellte Filter kann zur späteren Verwendung gespeichert werden. Speichern einer benutzerdefinierten Suche in MetraView

Anzeigen eines gespeicherten Berichts

Als MetraView-Benutzer bevorzugen Sie eventuell eine Möglichkeit, Nutzungsinformationen zu Ihren Gebühren oder den Gebühren der von Ihnen verwalteten Konten anzuzeigen. Wenn Sie eine solche Ansicht speichern, wird diese unter Meine Berichte angezeigt.

In einem gespeicherten Bericht bleiben die Such- und Filterwerte erhalten, die Sie zuvor für die Ansicht von Informationen festgelegt haben. Es wird jedoch nicht der Zeitraum gespeichert, für den Sie diese Informationen angezeigt haben. Wenn Sie einen gespeicherten Bericht öffnen, ist es häufig sinnvoll, eine Rechnungsperiode auszuwählen, die vor der aktuellen Periode liegt. (Je nach den Einstellungen für Ihre Rechnungen liegen in der aktuellen Periode eventuell keine Nutzungsdaten vor.)

Beim Speichern eines Berichts werden keine Ergebnisse gespeichert. Sie können verfolgen, wie sich Ihre Nutzung von Rechnungsperiode zu Rechnungsperiode verändert und diese mit den gleichen Anzeigekriterien vergleichen.

Speichern eines Berichts

Wenn Sie über Nutzungsanmeldungen verfügen, können Sie eine persönliche Datenansicht erstellen und diese wiederverwenden. In einem gespeicherten Bericht können Sie Such- und Filterwerte zur späteren Wiederverwendung speichern. Diese Suchen werden als Berichte bezeichnet und auf der MetraView-Übersichtsseite im Bereich Meine Berichte sowie auf der Registerkarte Meine Berichte angezeigt.

Beim Speichern eines Berichts werden keine Ergebnisse gespeichert.

Sie können verfolgen, wie sich Ihre Nutzung von Rechnungsperiode zu Abrechnungsperiode verändert und diese mit den gleichen Suchkriterien vergleichen.

So speichern Sie einen Bericht:

  1. Richten Sie die gewünschten Such- und Filteroptionen so ein, wie Sie sie speichern möchten.
  2. Wählen Sie in der rechten oberen Ecke des Such- bzw. Filterbereichs der Tabelle das Symbol Suche speichern.

    Wenn in der Tabelle kein solches Symbol angezeigt wird, kann in dieser Tabelle keine Suche gespeichert werden.

  3. Geben Sie einen Namen und eine Beschreibung für Ihre Suche ein.

  4. Klicken Sie auf OK.

    Die gespeicherte Suche ist über "Meine Berichte" auf Ihrer Startseite oder der Seite mit Ihren Berichten unter “Nutzung“ verfügbar (beim nächsten Besuch der Seite).

Löschen von gespeicherten Suchen

So löschen Sie einen Bericht:

  1. Wählen Sie in der rechten oberen Ecke des Such- bzw. Filterbereichs der Tabelle das Symbol Gespeicherte Suche laden.

  2. Klicken Sie auf das rote X neben der zu löschenden Suche.

  3. Bestätigen Sie den Löschvorgang.

Anzeigen eines freigegebenen Berichts

Möglicherweise haben Sie Zugriff auf Verknüpfungen mit speziellen Berichten, die von Ihrem Unternehmen oder der Kostenstelle für Sie eingerichtet wurden. Diese werden im Bereich Freigegebene Berichte angezeigt.

Freigegebene Berichte können mit anderen Benutzern in Ihrem Unternehmen oder Ihrer Kostenstelle gemeinsam verwendet werden. Sie enthalten jegliche Art benutzerdefinierter Daten. Wenn unter den freigegebenen Berichten keine Berichte aufgeführt sind, können derzeit keine angezeigt werden.
