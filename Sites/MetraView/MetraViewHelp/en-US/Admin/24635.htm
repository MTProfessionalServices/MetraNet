Branding a Site

The branding process relies on standard style sheets and images, which are easily accessed and modified as described in the previous topic.

A simple example of branding using the supplied Red theme and a different logo image

To brand a site:

  • Select or modify a supplied CSS style sheet or create a new one.

    Stylesheets are located in RMP\Extensions\MetraView\Sites\<Site>\Styles

    The name of the Theme in site properties must match the name of the CSS file name.
  • Replace the logo image and images for backgrounds, column rectangles, and buttons.

    All images are located in RMP\Extensions\MetraView\Sites\<Site>\Images

Restart IIS and log in again to see the changes.
