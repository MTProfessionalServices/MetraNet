Exportar datos a un archivo CSV

Para exportar los datos:

  1. Filtre los datos que desea exportar en MetraView o MetraCare.
  2. Seleccione Exportar en la parte inferior de la cuadrícula.
  3. Seleccione una de las siguientes alternativas:
    • Exportar solo la página actual
    • Exportar todos los registros
  4. Seleccione Exportar.

    Si está exportando un conjunto de datos donde existen datos secundarios, deberá seleccionar los datos secundarios desea incluir en la exportación. Si está exportando a Excel, los datos secundarios se incluyen en fichas separadas. Si está exportando un archivo plano, se utiliza un archivo separado para cada archivo secundario.

  5. Seleccione si desea abrir el archivo resultante o si desea guardarlo. Los datos se guardan en formato CSV.
