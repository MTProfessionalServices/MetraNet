Filtrage, recherche et tri des données dans une facture en ligne

Vous pouvez modifier l’affichage des données qui apparaissent quand vous ouvrez les fenêtres Utilisation, Historique des paiements ou Ajustements dans MetraView. Si les données s’affichent dans une grille, la fenêtre Configurer des filtres vous permet de modifier les propriétés qui apparaissent dans la recherche et les filtres dans la page.

Pour configurer la recherche et les filtres :

  1. Sélectionnez l’icône Configurer des filtres située dans l’angle supérieur droit de la section de recherche ou des filtres dans la grille.

    Les options de configuration s’affichent.

  2. Cochez les éléments à inclure dans le filtre. Si une propriété ne peut pas être désélectionnée, c’est qu’elle a été configurée pour apparaître systématiquement dans les recherches ou les filtres.

  3. Cliquez sur OK.

    Après le filtrage, la recherche et le tri, vous pouvez enregistrer ce mode d’affichage des données afin de l’utiliser avec d’autres périodes et données de facturation. Enregistrement d’une recherche personnalisée dans MetraView
