Sélection d’un compte de facturation géré en ligne

Si vous êtes le responsable de facturation d’autres comptes, il est possible que vous ayez l’autorisation d’afficher leurs factures dans MetraView. Dans ce cas, lorsque vous ouvrez une session, une liste contenant un ou plusieurs noms de comptes de facturation gérés s’affiche.

Pour afficher votre propre compte :

  • Sélectionnez l’onglet Accueil ou un autre onglet.

Pour afficher un compte de facturation géré :

  1. Sélectionnez l’onglet Accueil.
  2. Choisissez un compte dans la liste Sélectionner une facture.

    Vous ne savez pas exactement quel compte vous êtes en train de consulter ?

    Vérifiez toujours le nom du compte affiché.

Si vous modifiez d’autres comptes, vos informations utilisateur sont enregistrées dans les journaux d’audit. Par exemple, si vous modifiez le compte de Christelle, les informations d’audit enregistrent "Admin Assistant logged in as Christelle".

Pour ouvrir une session avec votre nom d’utilisateur ou pour afficher un autre compte de facturation géré, vous devez d’abord fermer la session.
