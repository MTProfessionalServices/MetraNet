Sélection d’une méthode de facturation

Il existe trois options de facturation standard :

  • Aucun(e) : aucune facture PDF n’est créée pour votre compte.
  • Standard : une facture standard contenant le résumé de vos frais est enregistrée au format PDF et est disponible dans la section Mes rapports de MetraView.
  • Détails : une facture détaillée contenant des informations détaillées sur vos frais est enregistrée au format PDF et est disponible dans la section Mes rapports de MetraView.

Pour changer le type de facture préparée :

  1. Dans MetraView, sélectionnez Infos du compte » Méthode de facturation.
  2. Dans Facture papier, choisissez le type de facture que vous voulez trouver dans votre widget Mes rapports, puis cliquez sur OK.
