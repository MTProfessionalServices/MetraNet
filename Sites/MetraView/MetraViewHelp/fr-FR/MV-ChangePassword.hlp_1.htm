Réinitialisation et modification d’un mot de passe


Modification d’un mot de passe

Pour modifier votre mot de passe MetraView :

  1. Dans MetraView, sélectionnez Infos du compte » Modifier le mot de passe.
  2. Tapez votre mot de passe actuel ainsi que le nouveau, puis cliquez sur OK.

Utilisez ce nouveau mot de passe lors de votre prochaine ouverture de session.

En fonction de la sécurité implémentée sur votre site, il est possible que vous deviez modifier un mot de passe provisoire lors de l’ouverture de session.

  1. Tapez votre ancien mot de passe.
  2. Tapez le nouveau mot de passe, et retapez-le, puis cliquez sur OK.

À propos des mots de passe robustes

Un mot de passe robuste doit être difficile à deviner. Les critères d’un mot de passe robuste peuvent être modifiés par votre administrateur. Par défaut, les caractéristiques d’un mot de passe robuste sont les suivantes :

  • il doit comporter au moins 7 caractères ;
  • il doit contenir au moins une lettre minuscule, une lettre majuscule et un chiffre ;
  • l’usage de caractères spéciaux est facultatif.

À propos des mots de passe uniques

Lorsque vous modifiez votre mot de passe, vous devez en choisir un nouveau qui soit unique. Un mot de passe unique est un mot de passe que vous n’avez encore jamais utilisé. En règle générale, vous devez utiliser un nombre de mots de passe minimum avant de pouvoir en répéter un. Ce nombre est défini par l’administrateur. Par défaut, vous devez employer 4 mots de passe uniques avant de pouvoir en répéter un.

Ouverture d’une session après blocage

Votre connexion à MetraNet peut être bloquée pour différentes raisons :

  • oubli du mot de passe ;
  • trop grand nombre de tentatives d’ouverture de session ;
  • inactivité du compte ;
  • expiration du mot de passe.

Si votre connexion à MetraNet est bloquée, vous devez contacter votre administrateur pour réinitialiser votre mot de passe. L’administrateur communiquera le nouveau mot de passe à utiliser pour ouvrir une session. Une fois la session ouverte, vous devez modifier ce mot de passe.
