Effectuer un paiement dans MetraView

Pour effectuer un paiement dans MetraView :

  1. Dans MetraView, sélectionnez Facture & paiements » Effectuer un paiement.
  2. Dans Montant du paiement, choisissez de payer le montant total dû ou une partie seulement.
  3. Dans Méthode de paiement, choisissez de payer à partir de votre compte bancaire ou par carte de crédit. Si vous avez déjà ajouté une méthode de paiement, sélectionnez-la dans la liste déroulante Méthode de paiement existante.
  4. Dans Date de paiement, choisissez si vous voulez payer la facture immédiatement ou à une date ultérieure.
  5. Cliquez sur Suivant. Un résumé s’affiche.
  6. Vérifiez que les informations sont correctes, puis cliquez sur Suivant. La page de confirmation apparaît.
