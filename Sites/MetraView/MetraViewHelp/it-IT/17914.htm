Esportazione di dati in un file CSV

Per esportare i dati:

  1. In MetraView o MetraCare, filtrare i dati che si desidera esportare.
  2. In fondo alla griglia, selezionare Esporta.
  3. Selezionare una delle seguenti opzioni:
    • Esporta solo pagina corrente
    • Esporta tutti i record
  4. Selezionare Esporta.

    Se si sta esportando un insieme di dati contenente dati figli, è necessario selezionare i figli che si desidera includere nell'esportazione. Se si sta esportando in un file di Excel, i dati figli verranno inclusi in schede separate. Se si sta esportando in un file flat, verrà utilizzato un file separato per ogni figlio.

  5. Scegliere se il file risultante verrà aperto o salvato. I dati vengono salvati in formato CSV.
