Reimpostazione e modifica della password


Modifica della password

Per cambiare la password MetraView:

  1. In MetraView, selezionare Info account » Modifica password.
  2. Digitare la password corrente e la nuova password e fare clic su OK.

Utilizzare la nuova password al successivo accesso.

In base alla sicurezza del sito, è possibile che sia necessario impostare una password temporanea dopo l'accesso.

  1. Digitare la vecchia password.
  2. Digitare la nuova password e confermarla, quindi fare clic su OK.

Informazioni sulle password complesse

Una password è detta complessa quando è difficile da indovinare. L'amministratore può cambiare i requisiti di complessità delle password. Per impostazione predefinita una password complessa:

  • deve essere composta da almeno 7 caratteri
  • deve contenere almeno una lettera minuscola, una lettera maiuscola e una cifra
  • i caratteri speciali sono opzionali

Informazioni sulle password univoche

Durante la modifica della password occorre impostare una nuova password univoca. Una password univoca è una password che non è stata utilizzata in precedenza. Solitamente, è necessario avere utilizzato un certo numero di password diverse prima di poter riutilizzare una password. Il numero di password univoche viene gestito dall'amministratore. Per impostazione predefinita, è necessario avere utilizzato almeno 4 password univoche prima di poter impostare una password già utilizzata in precedenza.

Accesso dopo il blocco del conto

Il conto MetraNet può venire bloccato per diversi motivi:

  • password dimenticata
  • troppi tentativi di accesso
  • inattività del conto
  • password scaduta

Se il conto MetraNet è bloccato, è necessario contattare l'amministratore per reimpostare la password. L'amministratore fornirà all'utente una nuova password da utilizzare per l'accesso. Una volta eseguito l'accesso, sarà necessario cambiare la password.
