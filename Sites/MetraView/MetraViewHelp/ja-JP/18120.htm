Editing the Dashboard

The Dashboard contains a collection of Columns, Widgets, and Parameters. The Dashboard is therefore a container for the objects and properties defined by these other entities. The Dashboard, itself, is contained by the Site.

To edit the Dashboard:

  1. From Configure Site, select the instance of the site you want to work with and click Edit.
  2. Under More Items, click the Dashboard component.
  3. Select the Dashboard instance you want to work with and click Edit.

Dashboard properties

Parameter

Description

Name

A unique name of the dashboard

Description

A description of the dashboard

Title

The title of the dashboard that may appear at the top.

CssClass

An override to CSS class used for the Site theme

Style

An additional set of styles to be added to the rendered dashboard.

Dashboard-contained components

BME

Purpose

Columns

Sets the columnar layout of the Dashboard.

Widgets

Sets widgets for each Column.

Parameters

Sets optional parameters for widgets.
