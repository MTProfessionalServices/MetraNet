Configuring a Site

The Configure Site menu item is your single point of access to all the configurable components of a MetraView site's user interface (UI). Starting here, you can add, edit, or remove any site component.

A Site is a nested collection of user interface controls contained in an instance of the MetraView web framework. When you drill down through a site you are selecting the control type and then one instance of that control.

For example, to access a particular Dashboard, you do the following:

  1. Open the Site you want to edit (there may only be one).
  2. Open the Dashboard collection belonging to that Site (again, there may just be one).
  3. Open the Dashboard instance you want to edit.

    User interface controls or components are called Business Modeling Entities (BMEs) elsewhere in MetraNet documentation for system integrators and configuration developers. For clarity, this help file refers to them simply as components.

Main Site components

Component

Purpose

BillSetting
