Editing the Site Components

The Site is a container for the various components that comprise a MetraView site's user interface. From the Site instance, you can create, edit, and delete instances of these components. You can access all the properties of these individual components through the Site container.

All component changes should be thoroughly tested before going live with the Site.

To edit a Site component:

  1. From Configure Site, select the instance of the site you want to work with and click Edit.
  2. Under More Items, click the component you want to edit.

    Component

    Purpose

    BillSetting

    Sets MetraView online bill properties.

    Dashboard

    Contains the Column and Widget components.

    EntryPoints

    Sets a nonstandard entry point to the site.

    ProductViewMappings

    Sets the display of usage information for a particular service.

    ReportInventories

    Manages the subscriber report types.

For general editing instructions, see Adding, Editing, or Deleting Components
