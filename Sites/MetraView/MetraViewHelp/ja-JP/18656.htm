BillSettings

The BillSetting component has properties that determine the billing interface for the whole site.

Parameter

Default Value

Description

Allow Online Payment

True

Permits subscribers to make payments online.

Allow Saved Reports

True

Permits subscribers to save their own filters as a report.

Allow Self-Care

True

Permits subscribers to update their contact and account information.

Inline Tax

False

Sets whether or not tax is included in amounts or broken out separately.

Inline Adjustments

False

Sets whether or not adjustments are shown inline or broken out separately.

Show Second-Pass Data

False

Shows aggregate billing data if an estimate is provided for first run.

MetraView currently supports a single instance of this component.
