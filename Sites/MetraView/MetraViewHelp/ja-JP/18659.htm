ReportInventories

MetraView supports four report types: Interactive Reports, Query-Generated Reports, Saved Filtered Grid Data, and Pre-generated PDFs. In the ReportInventory component, you can set various configuration parameters that will determine what criteria subscribers can use to search for reports There are no default values for these parameters.

Parameter

Description

Name

Name of the report

Description

Extended description of the report

ReportType

Indicates the type of report: Grid or Chart.

LayoutName

The internal name of the grid or chart layout to use for rendering.

Application

Name of the MetraNet application for which this report is for

Capabilities

Comma separated list of capabilities that the end users need to view this report

Category

Optional category used to group reports.

Priority

Numeric indicator specifying the priority ID of this report, so that it appears in the correct position in the report list

Visible

Indicates if the report is visible in the system.

UserId

AccountID of the user who created this report. For the reports created by SI's, User ID will be NULL
