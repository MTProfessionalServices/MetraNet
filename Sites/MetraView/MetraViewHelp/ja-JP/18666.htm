Widgets

A Widget is any ASCX user control. They are contained in MetraView\UserControls. You can add and remove widgets from the columns of your Dashboard.

To drill down to widgets:

  • Select Site » Dashboard » Column » Widgets

To add a new widget:

  1. Click Add.

  2. Enter the mandatory fields: name, position, and path to ASCX file.

    For example:

    Name

    BillAndPayments

    Position

    1

    Widget Path

    UserControls\BillandPayments.ascx

Each widget has a set of optional, configurable properties as well.

Parameter

Description

CssClass

An override to CSS class used for the Site, Dashboard, and Column themes.

Description

A description of the widget

Height

Set a fixed height in pixels (preserving aspect ratio).

Name

A unique name for the widget

Position

The position of the widget in the column (1, 2, 3 for 1st, 2nd, 3rd, and so on).

Style

Additional set of styles to be added to the rendered widget.

Title

The title of the widget that may appear at the top.

WidgetPath

The relative path from the Site folder (UserControls/<widget>.ascx)
