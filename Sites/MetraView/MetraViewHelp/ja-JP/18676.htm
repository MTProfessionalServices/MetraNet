Adding, Editing, or Deleting Components

Site components are potential interface items until you add them and configure them with defining properties. When you add a component, you are creating an instance of that component with a particular set of attributes: such as page location, column position, size, color, and so on. Sometimes, you can add multiple component instances to one page or several pages in your web application. For example, a particular widget, such as calendar, might be useful on several pages.

To open a component:

  • Select a component from a container component.

    For example, from Edit Site, you can access all the second-level components that belong to the site: BillSetting, Dashboards, EntryPoints, ProductViewMappings, and ReportInventories.

To modify an existing component:

  1. Select the instance and click the Edit icon.

  2. Proceed to set the component properties.

To create a component instance:

  1. In the Search Results window click Add.

  2. Proceed to set the component properties.

To delete a component:

  • Select the instance and click the Delete icon.

Deleting a component can have serious consequences. Never do this in a production environment.
