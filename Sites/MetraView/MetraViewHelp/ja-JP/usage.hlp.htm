自分の報告書

[自分の報告書] ページには、申し込まれているご利用プランおよび特定の期間中の利用料金のリストが表示されます。企業向けの請求書の場合は、担当しているグループ内の個人の利用状況を表示することもできます。これを、階層ビューと呼びます。

申し込みの利用状況を [集計] ビューまたは [詳細] ビューに切り替えることができます。

  • 集計ビューには、申し込んだサービスごとに利用料金の合計が表示されます。
  • 詳細ビューには、自分が支払者となっている、階層内のその他のアカウントの利用状況が表示されます。

利用状況の内訳を表示するには:

  1. MetraView で [自分の報告書] タブを選択します。現在の利用状況に関する報告書が表示されます。左側の列の [利用状況報告書] ウィジェットには、それ以外の利用状況報告書のリストが表示されます。
  2. 期間を変えて表示するには、[次の期間の利用状況を表示] で別の開始日と終了日を選択し、[移動] をクリックします。

    選択した期間の利用状況の概要が表示されます。

  3. 「詳細ビュー」 または 「集計ビュー」 をクリックすると、料金の表示を切り替えることができます。

    他の利用者 (たとえば、部門または他の原価部門の他の利用者) を管理する場合、料金の詳細ビューには、各支払アカウントごとに利用状況が表示されます。
  4. ご利用プランの計算ごとに料金を選択します。料金の左にあるエキスパンダを選択すると、これらの料金に関する内訳をさらに表示できます。
  5. 利用状況報告書に戻るには、[自分の報告書] タブを選択します。

    利用状況の詳細を表示する場合、フィルタを使用して結果を絞り込むことができます。その後、フィルタを保存して次回に使用することができます。カスタム検索の MetraView への保存

保存済み報告書の表示

MetraView ユーザは、自分の利用料金や自分が管理するアカウントの利用料金に関する情報を自分の望む方法で表示することができます。このビューを保存すると、ビューは [自分の報告書] に表示されます。

保存済み報告書には、情報を表示するために以前設定した検索値とフィルタ値が記憶されます。ただし、この情報の表示対象期間は記憶されません。保存済み報告書を開くときに、現在の請求期間よりも前の期間を選択することが必要になる場合があります (請求書の設定によっては、現在の期間に表示可能な利用データがない場合があります)。

報告書を保存してもその結果は保存されません。請求期間から請求期間の間で利用状況が変わった場合、同じ表示条件を使用して利用状況を比較できます。

報告書の保存

利用者は、データの個別のビューを作成し、再利用できます。保存済み報告書を使用すると、検索値とフィルタ値を保存して後で再利用できます。このような検索は、呼び出される報告書であり、MetraView Dashboard および [自分の報告書] タブにある [自分の報告書] 領域に保存されます。

報告書を保存してもその結果は保存されません。

請求期間から請求期間の間で利用状況が変わった場合、同じ検索条件を使用して利用状況を比較できます。

報告書を保存するには:

  1. 検索とフィルタを保存するときに、それらのオプションを設定します。
  2. グリッドの検索セクションまたはフィルタ セクションの右上にある [検索情報の保存] アイコンを選択します。

    このアイコンがグリッドにない場合、そのグリッドでは検索を保存できません。

  3. 検索の名前と説明を入力します。

  4. [OK] をクリックします。

    保存済み検索は、自分の [ホーム] または [自分の報告書の利用状況] ページ (次にページに移動する場合) の [自分の報告書] から使用できます。

保存済み検索の削除

報告書を削除するには:

  1. グリッドの検索セクションまたはフィルタ セクションの右上にある [保存済み検索情報の読み込み] アイコンを選択します。

  2. 削除する検索の隣にある赤い x 印をクリックします。

  3. 削除の確認を行います。

共通の報告書の表示

会社または原価部門が設定した、ユーザ向けの特別な報告書へのリンクが用意されている場合があります。このリンクは、[共通の報告書] セクションに表示されます。

共通の報告書とは、自分の原価部門または会社内の同僚と共有する報告書です。この報告書には、あらゆる種類のカスタム データが含まれます。[共通の報告書] に報告書が表示されていない場合、現在表示できる報告書はありません。
