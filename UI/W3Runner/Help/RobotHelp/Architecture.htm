Architecture

W3Runner is a Windows application that:

  • Embeds a Microsoft® Internet Explorer browser.

  • Exports Visual Basic Like COM objects to pilot the HTML objects from a VBScript file. What we intend by Visual Basic Like is that the methods and properties of the HTML objects are named and behave like the Visual Basic Controls as much as possible.

Objects :

' Simplified Syntax.
Page.Controls("txtFirstName").Text = "Freddy"
Page.Controls("chkTaxed").Value = vbChecked
 
' Generated Syntax.
Page.Controls("txtFirstName","TextBox").Text = "Freddy"
Page.Controls("chkTaxed","CheckBox").Value = vbChecked

Methods :

Page.Controls("txtFirstName").SetFocus
Page.Controls("cmdOK").Click

Properties : Name, Text, Value, List, ListCount, ListIndex.

MsgBox Page.Controls("chkTaxed").Value

Visual Basic functions turned into methods in the W3Runner interface : SendKeys.

Page.Controls("txtFirstName,"TextBox").SendKeys "Freddy"

Non Visual Basic methods and properties : AutoFireEvent, FireEvent, TypeAttribut, Object.

Note that the Object property returns the Microsoft Internet Explorer HTML object, the property Page.HTMLDocument returns the Microsoft Internet Explorer HTMLDocument object and the property Page.WebBrowser returns the Web Browser Control . Therefore, it is possible to have access to all the objects, methods and properties supplied by the Microsoft Internet Explorer Object Model if this is required.
