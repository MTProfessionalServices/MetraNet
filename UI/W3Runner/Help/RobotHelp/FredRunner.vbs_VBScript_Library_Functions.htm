W3Runner.vbs VBScript Library Functions

Include()

PUBLIC FUNCTION Include(strIncludeFileName) ' As Boolean

Include the VBScript source code in the main script.

IsValidObject()

PUBLIC FUNCTION IsValidObject(obj) ' As Boolean

Return TRUE is obj is an object and is different from nothing. The VBScript built-in function IsObject returns TRUE even is the value is Nothing.

IIF()

PUBLIC FUNCTION IIF(booExp,varRetValueTrue,varRetValueFalse) ' As Variant

Implement the Visual Basic IIF() function in VBScript.

IsBoolean()

PUBLIC FUNCTION IsBoolean(varV) ' As Boolean

Return TRUE is varV is a valid boolean value. Valid boolean values are : TRUE, FALSE, 0, 1, any other number. Remember the value 2 is TRUE but the value Not 2 is TRUE as well.

GetVBErrorString()

PUBLIC FUNCTION GetVBErrorString() ' As String

Return in a string the current error number, description and source.
