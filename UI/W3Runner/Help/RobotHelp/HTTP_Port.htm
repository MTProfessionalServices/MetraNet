Port

Public Property Get Port() As Long
Public Property Let Port(ByVal vNewValue As Long)

Description:
Get or set the port to redirect the HTTP protocol. This property allow ine to redirect the HTTP protocol to a specific port so the HTTP protocol can be traced with a specific tool.
This property also affects the property Page.URL and function Page.Post()

Sample:

strURL = "http://www.mydotcom.com/default.htm"
Http.UserName = "Rene1"
Http.PassWord = "RTR%^!323jUIA"
If Http.Execute("GET",strURL,,varText,,DOWNLOAD_TIME_OUT) = CHTTP_ERROR_OK Then
 
End If
 
HTTP.Port = 82
Page.URL = "http://Localhost/default.asp"
' The 2 previous lines behave like : Page.URL = "http://Localhost:82/default.asp"
 
HTTP.Port = 8080
Page.URL = "http://Localhost:81/default.asp"
' The 2 previous lines behave like : Page.URL = "http://Localhost:8080/default.asp"
