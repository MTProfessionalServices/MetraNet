StrConv

Public Function StrConv(varString As Variant, Conversion As VbStrConv, Optional LocaleID As Long) As Variant
 
' Visual Basic Enum Type VBStrConv
Public Const vbFromUnicode = 128 ' (&H80)
Public Const vbHiragana = 32 ' (&H20)
Public Const vbKatakana = 16 ' (&H10)
Public Const vbLowerCase = 2
Public Const vbNarrow = 8
Public Const vbProperCase = 3
Public Const vbUnicode = 64 ' (&H40)
Public Const vbUpperCase = 1
Public Const vbWide = 4

Description:
Returns a Variant (String) converted as specified.


Part

Description

varString

 String expression to be converted.

Conversion

The sum of values specifying the type of conversion to perform.

LocaledId

The LocaleID, if different than the system LocaleID. (The system LocaleID is the default.)

 
Remark:
When you're converting from a Byte array in ANSI format to a string, you should use the StrConv function. When you're converting from such an array in Unicode format, use an assignment statement.

Sample:

  Dim strVBSFile
 
  HTTP.URL = "http://1.2.3.4/W3Runnerdemo/PageValidationDemo.vbs"
  HTTP.Method = "GET"
 
  If HTTP.Execute(,,,strVBSFile) = CHTTP_ERROR_OK Then
 
  Msgbox Typename(strVBSFile)
  strVBSFile = W3Runner.StrConv(strVBSFile,vbUniCode)
  Msgbox Typename(strVBSFile)
  Msgbox strVBSFile
  Else
  W3Runner.TRACE "HTTP call failed " & HTTP.Status & " " & HTTP.StatusText
  End If
