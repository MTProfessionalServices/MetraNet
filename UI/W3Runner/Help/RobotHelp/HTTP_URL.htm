URL

Public Property Get URL() As String
Public Property Let URL(vNewValue As String) As String

Description:
Get or set the default URL for the next calls.

Remarks:
Each time you set the URL the collection of Parameters and InputHeaders is cleared and the property data is reset to an empty string.

Sample:

Http.URL = "http://www.mydotcom.com/default.htm"
Http.Method = "Get"
Http.UserName = "Rene1"
Http.PassWord = "RTR%^!323jUIA"
