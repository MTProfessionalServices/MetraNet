InputHeaders

Public Property Get InputHeaders() As

Description:
Return a collection of CVariable. This class is used to stored the input Headers.

Sample:

Http.Method = "POST"
HTTP.Url = "http://" & LOCAL_HOST & "/SOAP/MTSoapListener.asmx"
Http.Data = strSoap
HTTP.InputHeaders.Add "Content-Type","txt/xml"
HTTP.InputHeaders.Add "Content-Length",len(strSoap)
HTTP.Execute
