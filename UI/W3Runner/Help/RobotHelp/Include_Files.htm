Include Files

W3Runner.vbs VBScript standard script.

W3Runner includes some VBScript built-in const, classes and functions that are loaded from the file W3Runner.vbs located in the install folder. The file is documented in the section W3Runner.vbs VBScript Library, later in this document.

About re-using VBScript libraries

The function Include() includes the file at the end of the main script file, so all the const, classes and function become available. Once included the file is also available in the debugger.

Sample:

Include Page.ScriptFilePath & "\Test.Lib.vbs"
Include W3Runner.Environ("VBSCRIPT_TEST_LIB") & "\Test.Lib.vbs"
Include "C:\MyVBScriptLib\Test.Lib.vbs"

A W3Runner VBScript can contains VBScript const, class definitions.
