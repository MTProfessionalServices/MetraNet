Introduction

There are 5 main COM objects available in the VBScript file.

  • The global object Page As W3RunnerPage, allows one to program the current HTML page or frames The property Page.Windows() returns a collection of W3RunnerPage.

Page.URL = "http://localhost/myWebApp/default.asp"
Page.WaitForDownLoad "default.asp"
Page.Controls("aFindProduct","Anchor").Click

  • W3RunnerControl object that can only be returned by the property Page.Controls().

Page.Controls("LastName","TextBox").Text = LAST_NAME
Page.Controls("OK","Button").Click
MsgBox TypeName(Page.Controls("LastName","TextBox"))

  • The global W3Runner As W3RunnerAppllication Object expose some miscalleanous functions and properties related to the W3Runner application.

W3Runner.MsgBox "Hello"
If W3Runner.FAILED(MyFunction(),"MyFunction execution") Then Exit Function.

  • The golbal HTTP As CHTTP object allows one to execute HTTP calls.

If Http.execute("GET",strURL,, strSource)=CHTTP_ERROR_OK Then
End If

  • The global Report As W3RunnerReportBrowser Object is an Internet Explorer Browser used to display report information.

Report.URL = "c:\temp\MyReport.htm"
Report.WaitForDownLoad ".htm"
Report.Write "<B>Hello</B>"
