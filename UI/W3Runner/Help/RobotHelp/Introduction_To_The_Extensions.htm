Introduction To The Extensions

W3Runner Extensions are VBScript classes very usefull to write test script with VBScript. They are installed in the Extensions folder. There are more extensions installed with the product than the one documented here.

Example :

The extension CCSVFile allow one to read or write a CSV File, in a Object Oriented way. This extension is very usefull to write data-driven tests.

List :

  • CDB - VBScript class to execute SQL Queries based on ADO.

  • CCSVFile - VBScript class to read and write CSV file.

  • Collection Extension - The COM Object W3RunnerLib.CVBScriptExtensions method GetNewCollection() return a new instance of the Visual Basic object Collection.

  • CFIFO - Implement a First In - First Out object.

  • CRandomDataGenerator - VBScript class to generate different random boolean, string, long and double values.
