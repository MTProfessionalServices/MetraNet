Introduction to the W3Runner.vbs VBScript Library

The file W3Runner.vbs is located in the install folder. The file is loaded first with every script. The file set the mode Option Explicit so you are forced to declare your variables. The script export W3Runner constants, functions and classes.

Remark:
You must not change this file, but we recommend you to read it. It will give you a better understanding of the W3Runner Architecture
