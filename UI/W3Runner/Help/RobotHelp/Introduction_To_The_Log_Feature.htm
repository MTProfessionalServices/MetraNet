Introduction

All messages displayed in the Trace and Script Info list view are stored in the file %TEMP%\W3Runner.log.csv. To support the CSV syntax the char ',' is replaced with the char ';'.
You can also send message information to the Trace listview and into the file using the method W3Runner.TRACE().

Default Syntax:

Date Time , Line Type, The Message

Performance Syntax:

Date Time,[PERFORMANCE],Date,Time,Method,URL File Name,Full URL,Time waited for the page in ms

Sample:

9/6/2001 9:01:12 PM,[METHOD],Page.Controls(""aDemo1"";""Anchor"";""menu"")
9/6/2001 9:01:12 PM,[METHOD],aDemo1.Click
9/6/2001 9:01:12 PM,[METHOD],Page.WaitForDownLoad ""MyDemo.asp""
9/6/2001 9:12:51 PM,[PERFORMANCE],2001-09-06,21:12:51,Page.WaitForDownLoad,MyDemo.asp,http://MyApp/default.asp,2524

It is possible to load this file in a spread sheet or database and apply sorting and filtering. A Perfomance line provides more info, so it is possible to compute the average time for a specific page for a specific session. The only method monitored for performance is Page.WaitForDownLoad.

The menu Debug -> Trace -> Clear clears the listviews but not the file. The menu Debug -> Trace -> View open the log file with the default log file viewer (See option dialog General tab).
