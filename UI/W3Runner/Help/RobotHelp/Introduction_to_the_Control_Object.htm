Introduction

The object W3RunnerControl is not the Visual Basic Control object. But a W3RunnerControl implements some of the Visual Basic Controls methods and properties, so the programming is very similar. A W3RunnerControl object is returned by the property Page.Controls().

Sample:

' Simplified Syntax, the type is not defined
Page.Controls("LastName").Text = TEXT_LAST_NAME
' Standard syntax generated during the record mode.
Page.Controls("FirstName","TextBox").Text = TEXT_FIRST_NAME
Page.Controls("TaxExempt","CheckBox").Value = vbChecked
Page.Controls("Country","ComboBox").Text = TEXT_COUNTRY
 
Page.Controls("Country","ComboBox").Value = COUNTRY_CODE
Page.Controls("SubScriberType","RadioButton",,,"Gold").Value = TRUE

Remark:
When the type of the object is not defined in the call of the property Page.Controls(), the property looks up the object from the whole collection of HTML objects and therefore is slower.
On the other side, the syntax is more Visual Basic Like.
