AutoFire Missing JavaScript Events

In the Internet Explorer browser, when a user clicks on a combobox the JavaScript event OnChange is raised and when a user clicks on a radio button or check box event the event OnClick is raised. This behavior does not occured in a WebBrowser control, therefore W3Runner simulate it.

You can turn it off and raise the event yourself see FireEvent.
