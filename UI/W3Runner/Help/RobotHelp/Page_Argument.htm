Arguments

Public Property Get Arguments(
ByVal strName As String,
Optional ByVal varDefaultValue As Variant
) As String

Description:

Return the value of a W3Runner.exe command line argument. The W3Runner command line arguments syntax must be : W3Runner.exe -Name1 Value1 -Name2 Value2. If the name is not found the functions returns Empty.

Parameters:

Part

Description

strName

The name of the command line argument.

varDefaultValue

The value to be returned if the argument is not defined in the command line.

Sample: Command Line

W3Runner.exe -file "c:\mytests\test1.vbs" -function "main" -QuickMode "TRUE" -UserName "Travis"
 
Sample:
 
Select W3Runner.Arguments("-UserName")
  Case "Travis"
  Case "Chris"
End Select
 
If CBool(W3Runner.Arguments("-QuickMode")) Then
 
End If
