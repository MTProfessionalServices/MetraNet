W3Runner_OnError

PUBLIC SUB W3Runner_OnError(ByVal lngNumber, ByVal strDescription, ByVal strSource)
END SUB

Description:
If the function W3Runner_OnError is implemented, W3Runner will call the function if an un expected VBScript run time error occurs.

Parameters:

Part

Description

lngNumber

VBScript run time error number.

strDescription

VBScript run time error description.

strSource

VBScript run time error source.

Sample:

 ' Log in a special file all the run time error message
PUBLIC SUB W3Runner_OnError(ByVal lngNumber, ByVal strDescription, ByVal strSource)
 
  W3Runner.TextFile.LogFile W3Runner.Environ("TEMP") & "\W3Runner_OnError.log" , lngNumber & "," & strDescription & "," & strSource
END SUB
