RecordHTTPProtocol

Public Property Get RecordHTTPProtocol() As Boolean
Public Property Let RecordHTTPProtocol(ByVal vNewValue As Boolean)

Description:
Get or set if W3Runner should store the HTTP protocol of the script execution in the file %TEMP%\W3Runner.Http.Protocol.vbs.

Remarks:
The property can only be set programmatically and it is not persistant.

Sample:

W3Runner.RecordHTTPProtocol = TRUE
