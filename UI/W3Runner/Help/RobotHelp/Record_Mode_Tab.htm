Record Mode Tab

This tab groups global information about the record mode.

 

Part

Description

Record only from frame

When an application has a lot of frame it can be useful and faster to record only from one frame at a time.
For more information about the record mode limitation, see the ReleaseNotes.htm in the product or on the web site.

Watch Window Always On Top

Set the watch window, as always on top, of all other windows.
