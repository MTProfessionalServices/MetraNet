DownLoadTimeOut

Public Property Get DownLoadTimeOut() As Long
Public Property Let DownLoadTimeOut(ByVal vNewValue As Long)

Description:
Get or set the time out value for file Downloading . This applies to the method Page.WaitForDownLoad () and Report.WaitForDownLoad(). This property can be set in the options dialog as well.

Sample:

Page.DownLoadTimeOut = 60 ‘ One Minute TimeOut to download asp/html page
