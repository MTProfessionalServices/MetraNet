Report.WaitForDownLoad

Public Function WaitForDownLoad(ByVal strURL As String) As Boolean

Description:
Waits for the current URL to be fully downloaded.

Parameters:

Part

Description

strURL

The URL or piece of the URL to wait for. The case is ignored.

Remark:
You can set the time out with the property Page.DownLoadTimeOut or in the Options dialog.

Sample:

Page.URL = "http://www.W3Runner.com/default.asp"
Page.WaitForDownLoad “default.asp”
