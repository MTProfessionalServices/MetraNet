Test Report Extension

CTestReport - A VBScript class which extend the possibilities of the W3Runner Report object.

Interface :

CLASS CTestReport
  PUBLIC FUNCTION Initialize(strTitle)
  PUBLIC FUNCTION Summary()
  PUBLIC PROPERTY LET OutPutFolder(v)
  PUBLIC PROPERTY GET OutPutFolder()
  PUBLIC PROPERTY GET FileName()
  PUBLIC FUNCTION Save()
  PUBLIC FUNCTION WriteHTMLTable(lngCellPerRow,arrayValues)
  PUBLIC FUNCTION Show()
  PUBLIC FUNCTION Hide()
  PUBLIC FUNCTION Write(s)
  PUBLIC FUNCTION WriteLn(s)
  PUBLIC FUNCTION Clear()
END CLASS

Sample :

PUBLIC FUNCTION Main()
 
  Dim objReport
 
  Include W3Runner.Environ(w3rW3RUNNER_PATH) & "\Extensions\CTestReport.vbs"
  Set objReport = New CTestReport
  ' Initialize the report. The report title is the script file name
  objReport.Initialize Empty
 
  If MyFunction() Then
objReport.WriteLn "MyFunction() SUCCEEDED"
  Else
objReport.WriteLn "MyFunction() FAILED"
Exit Function
  End If
 
  ' Force the report to be generated and loaded by W3Runner, this is optional
  objReport.Summary
 
  ' Save the report to a local file.
  objReport.Save
END FUNCTION
