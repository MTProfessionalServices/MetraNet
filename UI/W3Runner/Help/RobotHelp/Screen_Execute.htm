Execute

Public Function Execute(
ByVal strProgram As String,
Optional ByVal strParameters As String,
Optional ByVal eAppWinStyle As VbAppWinStyle = vbNormalFocus,
Optional ByVal booSynchronous As Boolean = False,
Optional ByRef varAppId As Variant,
Optional ByRef varAppHWND As Variant,
Optional ByVal lngWaitTime As Long = 1,
Optional ByVal strWindowClassName As String
) As Boolean
 
' Visual Basic enum type VbAppWinStyle, implemented in the W3Runner.vbs file.
Public Const vbHide = 0
Public Const vbNormalFocus = 1
Public Const vbMinimizedFocus = 2
Public Const vbMaximizedFocus = 3
Public Const vbMinimizedNoFocus = 6
Public Const vbNormalNoFocus = 4
 

Description:
Execute a program.

Parameters:

Part

Description

strProgram

The program to execute with or without path.

strParameters

The parameters

eAppWinStyle

Window display style

booSynchronouse

If TRUE wait for the end of the program. In synchronous mode the parameter varAppId, varAppHWND, lngWaitTime and strWindowClassName are ignored.

varAppId

Task id returned by the shell command

varAppHWND

The main window handle of the application

lngWaitTime

Specify the waiting time needed by the application lanched to initialize itself before the method can find the main HWND of the application. Needed only if varAppHWND parameter is defined.

strWindowClassName

If not defined the method returns the first HWND belonging to the process lanched. By defining the parameter you can specify the window class name of the window you want to be returned. This is useful with Visual Basic 6 applications which open more than one window at start time. See sample. Needed only if varAppHWND parameter is defined.

Sample:

W3Runner.Execute "Notepad.exe", "c:\myfile.txt",,TRUE
 
' Start a Visual Basic Application and make sure will will get the HWND of the main window.
W3Runner.Execute "c:\MyVB6App.exe",,,,,lngHWND,1,"ThunderRT6FormDC"
