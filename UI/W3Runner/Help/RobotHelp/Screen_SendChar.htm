SendChars

Public Function SendChars(
ByVal strChars As String,
Optional ByVal lngLPARAM As Long = 0,
Optional ByVal booPost As Boolean = True
) As Boolean

Description:
Send a string of chars to the current object. This method does not support any macros like the method SendKeys().

Parameters:

Part

Description

strChars

The chars to be sent.

lngLPARAM

The LPARAM value of the message. Reserved.

booPost

If TRUE use the Windows API PostMessage instead of SendMessage.

Remark:
This method use the Windows Message WM_CHAR to send each char. For the char TAB the Windows Message WM_KEYDOWN and WM_KEYUP are sent.

Sample:

W3Runner.SendChars "Smith"
