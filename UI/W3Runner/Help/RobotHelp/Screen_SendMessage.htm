SendMessage

Public Function SendMessage(
ByVal lngHWND As Long,
ByVal lngMESSAGE As Long,
ByVal lngWPARAM As Long,
ByVal lngLPARAM As Long,
Optional booPost As Boolean = False
) As Boolean

Description:

Call the SendMessage or PostMessage Windows API.

Parameters:

Part

Description

lngHWND

The Windows Handle to send the message. The property Page.WebBrowserHWND return the WebBrowser window handle.

lngMessage

The Windows Message.

lngWParam

The WPARAM value of the message.

lngLPARAM

The LPARAM value of the message.

booPost

If TRUE use the Windows API PostMessage instead of SendMessage.

Sample:

W3Runner.SendMessage Page.WebBrowserHWND,WM_CHAR,Asc("A"), 0
 
' You can call the api PostMessage this way
W3Runner.SendMessage Page.WebBrowserHWND,WM_CHAR,Asc("A"), 0, TRUE
