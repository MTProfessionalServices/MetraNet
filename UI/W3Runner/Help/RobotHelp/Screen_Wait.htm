Wait

Public Function Wait(
ByVal dblSecond As Double,
Optional ByVal booNoSleep As Boolean = False
) As Boolean

Description:
Waits for a number of second. The value can be less that one. If the value is greater than 1, after each tenth of second W3Runner yields execution so that the operating system can process other events. The Windows Api Sleep is used by this method.

Parameters:

Part

Description

dblSecond

The number of second to wait.

booNoSleep

If TRUE the function yields the execution for all the amount of time, the Windows Api Sleep is not used.

Sample:

W3Runner.Wait 0.25
W3Runner.Wait 3
 
CONST LOGIN_WAIT_TIME = 3
W3Runner.Wait LOGIN_TIME_WAIT_TIME
