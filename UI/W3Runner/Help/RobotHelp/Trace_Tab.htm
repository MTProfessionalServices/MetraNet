Trace Tab

This tab groups global information about the tracing feature.

 

Part

Description

Show an auto-close message box on error

If an error is is traced, an auto-close message box will pop up.

Trace Type Group

Gather all the types of traces that must be displayed in the Trace ListView and logged in the log file.

Script Info

Gather all the type of traces that must be displayed in the Script Info ListView and logged in the log file.

List View

Define if the list views must be cleared when they reach a certain number of rows. One can use this tab to avoid the application of too much much memory use.
