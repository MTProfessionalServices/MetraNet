Introduction

The object W3Runner.TextFile exports text file manipulations functions. The object W3Runner.Stack implements a Variant stack. The object CVariables class implements a collection of Name, Value, Type, Caption instance. The object CIniFile

The class CVariables and CIniFile must be created this way :

Set objVariables = CreateObject(CVariables)
Set objIniFile = CreateObject(CIniFile)
