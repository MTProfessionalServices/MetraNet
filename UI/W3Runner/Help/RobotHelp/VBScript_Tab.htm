VBScript Tab

This tab groups global information about the script execution .

 

Part

Description

Enable Debugger

Active the debugger for the next execution. You must set a break point with the key word STOP.

Move Mouse Cusor

Define if W3Runner must move the cursor above each object returned by the property Page.Controls(). . Use can use the keyboard macro CTRL+M to change this options. You can change this value programmatically with the property W3Runner.MouseMoveCursor.

Auto Focus

Define if W3Runner must set the focus to object returned by the property Page.Controls(). Setting the focus will force the WebBrowser window to scroll if the object is not visible.

Add error line information to the scripts

Add extra information to each line of the script to report better error information. This is specially usefull for included file.

SlowMode

Set the slow mode. See the property Page.SlowMode.
