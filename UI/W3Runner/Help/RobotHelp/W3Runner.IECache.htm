IECache

Public Property Get IECache() As CIECache

Description:
Return the global instance of the object IECache. The COM Object CIECache allow to manipulate the Internet Explorer cache.

Sample:

  ' ' Download some page, then load the IE Cache and display all the file names. ' Then delete all and check that the cache is empty '
 
  Page.URL = "http://www.w3runner.com" Page.WaitForDownLoad ".asp"
 
  ' Load the cache
 
  W3Runner.IECache.Load W3Runner.TRACE W3Runner.IECache.Count & " file(s) in the IE cache." ' Display the content of the cache
For Each objIECacheEntry In W3Runner.IECache W3Runner.TRACE objIECacheEntry.URL Next
' How to delete one file ' The pattern matching used is based on the Visual Basic operator LIKE.
W3Runner.IECache.DeleteAll "http://www.w3runner.com/header.asp" ' Delete all the file in the cache that contains the word w3runner in the
  ' path or the name. ' The pattern matching used is based on the Visual Basic operator LIKE.
W3Runner.IECache.DeleteAll "*w3runner*"
 
  ' We need to reload the cache at this point
  W3Runner.IECache.Load
