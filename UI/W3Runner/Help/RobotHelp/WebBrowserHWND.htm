WebBrowserHWND

Public Property Get WebBrowserHWND() As Long

Description:
Return the HWND of the embeded Web Browser window class "Internet Explorer_Server". This handle should be used to send Windows message to the Web Browser.

Sample:

W3Runner.SendMessage Page.WebBrowserHWND,WM_LBUTTONDOWN,67687
