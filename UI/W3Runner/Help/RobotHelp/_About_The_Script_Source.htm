About the test script

Public Methods
A W3Runner VBScript file must contains at least one public function which will be called from the W3Runner User Interface. The first public function available can be executed by pressing the F5 key or the Run icon in the tools bar. The other public functions can be executed through the menu Run -> Functions.

Classes and constants
A W3Runner script can contain VBScript constants, subs, functions and classes definitions.

W3Runner.vbs VBScript standard include
W3Runner supplies some built-in constants and functions which are loaded from the file W3Runner.vbs located in the install folder. The file is documented in the section W3Runner.vbs VBScript Library, later in this document.

How to Include Files
The global function Include() includes one file at the end of the main script file. All the constants, subs, functions and classes become available. Once included, the file is also available in the debugger.
You cannot include a file twice.

Debugging.
See chapter How to debug a script.

Sample:

Include Page.ScriptFilePath & "\Test.Lib.vbs"
Include W3Runner.Environ("VBSCRIPT_TEST_LIB") & "\Test.Lib.vbs"
Include "C:\MyVBScriptLib\Test.Lib.vbs"
 
Private Function IncludeALL()
 
Include W3Runner.Environ("VBSCRIPT_TEST_LIB") & "\MyLib.vbs"
End function
