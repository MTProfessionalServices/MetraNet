CommandLine

Public Property Get CommandLine() As CCommandLine

Description:
Return the global instance of the CCommandLine object. The object allow to get the different token of the command line.

Interface:

CLASS CCommandLine

Public Property Get CommandString() As String
Public Property Get Item(ByVal vntKey As Long) As String
Public Property Get Count() As Long
Public Function Exist(ByVal vntKey As Variant) As Long
Public Function GetValue(ByVal vntKey As Variant, Optional ByVal vntDefaultValue As Variant) As Variant

END CLASS

Sample:

Dim strS, i
For i = 1 to W3Runner.CommandLine.Count
 
  strS = strS & W3Runner.CommandLine.Item(i) & " "
Next
