CancelScript

Public Property Get CancelScript() As Boolean
Public Property Let CancelScript(v As Boolean)

Description:

Get or set if the user asked to cancel the script by pressing the stop icon in the toolbar. In the test script file, you should insert at the end of each loop the following code so it is possible to ask to cancel the test more rapidly.

If Page.CancelScript Then Exit function

Sample:

For i=1 To lngNumber
  Page.Controls("aEditCustomer","Anchor","menu").Click
  Page.WaitForDownLoad "EditCustomerDialog.asp"
  PopulateAndAddCustomer
  If W3Runner.CancelScript Then Exit For
Next
