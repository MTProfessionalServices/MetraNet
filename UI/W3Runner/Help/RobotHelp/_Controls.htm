Controls

Public Property Get Controls(
Optional ByVal strName As String,
Optional ByVal strType As String,
Optional ByVal strFrame As String,
Optional ByVal lngIndex As Long = 1,
Optional ByVal varValue As Variant,
Optional ByVal strHREF As String) As W3RunnerControl

Description:
Return the W3RunnerControl object that match the parameters.
Generally an HTML tag should have an attribute Name (or Id) which identify uniquely the object in the page and that should be used to look up the object. If the HTML Tag does not have any Name or Id attribute, W3Runner allow you to look up the object in a very powerfulll way, see Detailed look up method in the help page.

The Record mode will alway generate the best way to look up an object.

Parameters:

Part

Description

strName

Contains the name, the text or the file name of the object to return.

strType

Can be the following values : TextBox (HTML Tag <INPUT Type=’Text’> or <TEXTAREA>), RadioButton, CheckBox, Button, Input, Submit (Special button),
ComboBox, ListBox, Image, Anchor, TableCell (HTML Tag <TD>), Div (HTML Div), and empty string. If strType is not defined then the function will try to look up the object based
on the name or the text only with the optional lngIndex or varValue parameters.

strFrame

The name of the frame if the object is located in a frame. HTML object in a child window does not need frame name.

lngIndex

When an object is defined more than once with the same name for instance, you can specify the index to locate the object. This is generally used to located images with the same file name or
button with the same name.

varValue

When an object is an array it may have a HTML attributes value that makes it unique. This is how HTML Radio Button behave.

strHREF

If few anchor <A> have the same name you can use the HREF.

Remarks :

  • If the Mouve Move Cursor mode is on, this function will move the mouse cursor to center of the object returned. See the W3Runner.MoveMouseCursor property for more information.

  • If the Auto Focus mode is on, the focus will be set to object returned. See the Page.AutoFocus property for more information.

Sample:

‘ Look up a text box
Page.Controls("LastName","TextBox","main").Text = “Smith”
‘ Look up a button
Page.Controls("OK","Button").Click
‘ Look up a combo box
Page.Controls("Country","ComboBox","main").Value = "ITALY"
‘ Look up a check box
Page.Controls("TaxExempt","CheckBox","main").SetFocus
‘ Look up a radio button
Page.Controls("SubScriberType","RadioButton","main",,"Gold").Value = TRUE
‘ Look up anchor based on the text
Page.Controls("Add A New User","Anchor","main").Click
‘ Look up an button with a non unic name
Page.Controls("ButEdit","Button",,2).Click
‘ Look up an image
Page.Controls("Go.gif","Image","main").Click
‘ Look up an anchor based on the name and the href
Page.Controls("MyLink","Anchor","main",,,"MyPage.asp?param=A").Click
‘ Look up an anchor based on the href
Page.Controls(,"Anchor","main",,,"MyPage.asp?param=B").Click
‘ Look up a DIV
Page.Controls("MyDiv1","Div").Click
 

Detailed look up method :

If strType = “TEXTBOX” or “COMBOBOX” Then

  • Lookup the object based on the name and optionaly with the index if defined.

If strType = “RADIOBUTTON” or “CHECKBOX” or “SUBMIT” or “HIDDEN or “INPUT” Then

  • Lookup the object based on the name and optionaly with the index or the value if defined. It is mandatory to specify the value to identify a specific RadioButton.

  • If the object is not found try to look up the object based on the value and with the optional parameter index if defined.

If strType = “BUTTON” Then

  • Lookup the object based on the HTML Tag <INPUT>, the name and with the optional parameter index if defined.

  • If the object is not found try to lookup the object based on the HTML Tag <INPUT>, the HTML attribute value and with the optional parameter index if defined.

  • If the object is not found try to Lookup the object based on the HTML Tag <BUTTON>, the name and with the optional parameter index if defined.

  • If the object is not found try to look up the object based on the HTML Tag <BUTTON> the HTML attribute value and with the optional parameter index if defined.

If strType = “IMAGE” Then

  • Lookup the object based on the HTML Tag <IMG>, the HTML attribute src and with the optional parameter index if defined. The parameter strName may contain the all image file name or just a part.

  • If the object is not found try to look up the object based on the HTML Tag <IMG>, the HTML attribute name and with the optional parameter index if defined.

  • If the object is not found try to look up the object based on the HTML Tag <INPUT Type=’Image’>, the HTML attribute name and with the optional parameter index if defined.

If strType = “ANCHOR” Then

  • Look up the object based on the HTML Tag <A>, the text of the anchor and with the optional parameter index if defined. The parameter strName must contain the exact text, the space chars at the beginning and at the end are ignored.

  • If the object is not found try to look up the object based on the HTML attribute name and with the optional parameter index if defined.

  • If the object is not found try to look up the object based on the the HTML attribute HREF and with the optional parameter index if defined. You may only supply a piece of the HREF value.

If strType = “TABLECELL” OR “TD” Then

  • Look up the object based on the HTML Tag <TD>, the text of the table cell and with the optional parameter index if defined. The parameter strName must contain the exact text, the space chars at the beginning and at the end are ignored.

  • If the object is not found try to look up the object based on the HTML Tag <A>, the HTML attribute name and with the optional parameter index if defined.

If strType = “FRAME” Then

  • Look up the HTMLFrameElement based on the name. The search of the Frame object is not recursive therefore the frame must be defined in the current frameset level.

If strType = “HTMLWindow2” Then

  • Look up a frame based on the name but return the HTMLWindow2 interface rather than the HTMLFrameElement interface. The search of the Frame object is recursive.

If strType = “DIV” Then

  • Look up a DIV Tag based on the HTML attribute name..

  • If the object is not found try to look up the object based on its text.

If strType = “” Then

  • Lookup the object based on the HTML attribute name and with the optional parameter index if defined.

  • If the object is not found look up the object based the text of the HTML Tag if available and with the optional parameter index if defined. The parameter strName must contain the exact text, the space chars at the beginning and at the end are ignored.

  • If the object is not found look up the object based on the HTML attribute href and with the optional parameter index if defined.

  • If the object is not found a message box error is open, the error message is also logged. Since the object is not found the Controls() property will return Nothing, which will raise en VBScript Run Time Error. The Script will be canceled.
