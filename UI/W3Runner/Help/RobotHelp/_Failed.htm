Failed

Public Function Failed(
ByVal booExpression As Boolean,
ByVal strMessage As String
) As Boolean

Description:
This function allow to process the result of a unit test by triggering an action, diplaying and logging an error status.

Parameters:

Part

Description

booExpression

The boolean result of the unit test

strMessage

The message describing a unit test.
If booExpression is FALSE

  • The string strMessage will be displayed in the trace list as an operation that failed (Error) else as an operation that succeeded.

  • The string strMessage will be logged in the log file as an operation that failed (Error) else as an operation that succeeded.

  • The string strMessage will be displayed in error message box as an error. Optional according the Options Dialog.

Sample:

If(W3Runner.Failed(CreateNewCustomer(),”Function CreateNewCustomer()”)Then Exit Function
If(W3Runner.Failed(IsEmpty(lngUserID),"User ID Creation")Then Exit Function
