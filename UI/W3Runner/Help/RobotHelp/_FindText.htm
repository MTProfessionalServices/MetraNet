FindText

Public Function FindText(
ByVal strSearchedText As String,
Optional ByVal strFrameName As String,
Optional ByVal strTextType As String = "innerText",
Optional ByVal booIgnoreCase As Boolean = True,
Optional booRegularExp As Boolean = False
) As Long

Description:
Find and return the position of a text in the current HTML or Text of the page. If the text is not found the function returns 0. If the regular expression mode is set then the function return only TRUE or FALSE.

Parameters:

Part

Description

strSearchedText

The text to be searched.

strFrameName

The frame name if there are frames.

strTextType

Microsoft Internet Explorer DHTML property to be used in the search :InnerText, OuterText, InnerHTML, OuterHTML.

booIgnoreCase

Ignore case.

booRegularExp

Use regular expression

If Page.FindText("error.gif", “main”,"innerHTML") Then
 
End If
