Integration with other tools

W3Runner support the following features to interact with other tools :

  • You can execute a script from the command line. See chapter Command Line Syntax.

  • The W3Runner object supports the following events : W3Runner_OnError and W3Runner_OnTrace which allow you to execute some VBScript code in the following cases

    • Your test script has detected an error or warning

    • Your test script has raised an unexpected run time error

  • The method W3Runner.Quit allows to stop W3Runner and start another instance of W3Runner or another program.

The events are details in the Scripting Interface chapter, W3Runner Object.
