MoveMouseCursor

Public Property Get MoveMouseCursor() As Boolean
Public Property Let MoveMouseCursor(ByVal vNewValue As Boolean)

Description:
Get or set if W3Runner must move the mouse during the script execution. The property can also be set in the options dialog tab general.

Sample:

' Make sure that we do not move the cursor when we click on a div
Dim booMoveMouseCursor
booMoveMouseCursor = W3Runner.MoveMouseCursor
W3Runner.MoveMouseCursor = FALSE
Page.Controls("MyDiv1","Div").Click
W3Runner.MoveMouseCursor = booMoveMouseCursor
