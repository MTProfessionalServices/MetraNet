MsgBox

Public Function MsgBox(ByVal strMessage As String, Optional strTitle As String) As Boolean

Description:
Displays the message and the title in a message box. The message box will close automatically after a few seconds if nobody presses OK. The default time out value is 20 second.
The value can be set in the option dialog or with the property W3Runner.MsgBoxTimeOut.

Parameters:

Part

Description

strMessage

The message to be displayed.

strTitle

The title of the message box

Sample:

W3Runner.MsgBox "Time=" & Now()
