ByVal strURL As String,
Optional ByVal strTargetFrame As String,
Optional ByVal lngFlags As Long

Description:
Execute an HTTP POST using the webbrowser the result of the page is rendered.

Parameters:

Part

Description

strURL

The URL

strTargetFrame

The frame name.

lngFlags

Reserved.

Page.Parameters.Clear
