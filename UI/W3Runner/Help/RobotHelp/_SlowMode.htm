SlowMode

Public Property Get SlowMode() As Long
Public Property Let SlowMode(ByVal vNewValue As Long)

Description:
Get ot set the SlowMode. If the property value is different from 0. The property Page.Control() will wait for the SlowMode value amount of time before returning the object. The unit is milli-second.

The goal of the property Page.SlowMode, is to slow down the script execution, so you can follow the execution. When the script is perfectly synchronized that is to say all the pages downloaded and all the Page.WaitForDownLoad() match perfectly you should not need to set the slowmode on any machine.

Sample:

Page.SlowMode = 200
Page.SlowMode = 0
