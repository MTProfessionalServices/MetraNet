Public Function TRACE(ByVal strMessage As String,Optional ByVal eMode As eW3RunnerTraceMode = w3rINFO) As Boolean
 
PUBLIC CONST w3rINFO = 1
PUBLIC CONST w3rERROR = 2
PUBLIC CONST w3rWARNING = 4
PUBLIC CONST w3rSQL = 8
PUBLIC CONST w3rURL_DOWNLOADED = 16
PUBLIC CONST w3rMEMORY = 32
PUBLIC CONST w3rDEBUG = 64
PUBLIC CONST w3rINTERNAL = 128
PUBLIC CONST w3rMETHOD = 256
PUBLIC CONST w3rJAVASCRIPT = 512
PUBLIC CONST w3rSUCCESS = 1024
PUBLIC CONST w3rCLEAR_TRACE = 2048
PUBLIC CONST w3rPERFORMANCE = 4096

Description:
Displays the string in the Trace list view and logs the string in the file %TEMP%\W3Runner.log.csv. To log an error it is advised to use the method W3Runner.FAILED(). The following trace mode are reserved for W3Runner :

  • w3rURL_DOWNLOADED

  • w3rINTERNAL

  • w3rMETHOD

  • w3rJAVASCRIPT

  • w3rPERFORMANCE

For more information see the chapter Log File.

Parameters:

Part

Description

strMessage

Message to be displayed in the Trace list view and logged in the log file.

eMode

Trace type

Sample:

' Trace an info message
W3Runner.Trace "CRM Application Testing Starting At " & Now()
' Trace an warning
W3Runner.Trace "German translation not found", w3rWARNING
' Trace an error
W3Runner.Trace "Customer creation failed", w3rERROR

Log File Sample:

9/19/2001 4:21:05 PM,[INFO],CSR Application Testing Starting At 9/19/2001 4:21:05 PM
9/19/2001 4:21:05 PM,[WARNING],German translation not found
9/19/2001 4:21:05 PM,[ERROR],Customer creation failed
