URL

Public Property Let URL(ByVal vNewValue As Variant)

Description:
Set the URL to download. Remember that after you set the URL you must wait for the page to be fully downloaded before continuing.

Sample:

Page.URL = http://www.W3Runner.com/default.asp
Page.WaitForDownLoad “default.asp”
