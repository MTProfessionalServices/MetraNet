Public Property Get Windows() As W3RunnerPages

Description:
Return a collection of W3RunnerPage object that match the HTML pop up window open. Generally a new HTML pop up window is open with the JavaScript window.open().

The collection exposes the following method :

Public Property Get Item(ByVal lngIndex As Long) As W3RunnerPage
Public Property Get Count() As Long
Public Function Exist(ByVal vntKey As Variant) As Boolean

Remarks:
The Item method support only long key, there is no name set to a pop up window.

Sample:

Page.Windows(1).Controls("LastName","TextBox").Text ="OK"
