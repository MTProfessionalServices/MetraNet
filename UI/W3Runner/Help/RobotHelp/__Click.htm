Click

Public Sub Click(
Optional ByVal X As Long,
Optional ByVal Y As Long,
Optional eFlag As w3rITEM_CLICK_FLAG = w3rITEM_CLICK_FLAG_LEFT_CLICK + w3rITEM_CLICK_FLAG_WM_MOUSE_DOWN + w3rITEM_CLICK_FLAG_WM_MOUSE_UP
)
 
PUBLIC CONST w3rITEM_CLICK_FLAG_ABSOLUTE = 1
PUBLIC CONST w3rITEM_CLICK_FLAG_RIGHT_CLICK = 2
PUBLIC CONST w3rITEM_CLICK_FLAG_LEFT_CLICK = 4
PUBLIC CONST w3rITEM_CLICK_FLAG_CURRENT_POSITION = 8
PUBLIC CONST w3rITEM_CLICK_FLAG_WM_MOUSE_DOWN = 16
PUBLIC CONST w3rITEM_CLICK_FLAG_WM_MOUSE_UP = 32

Description:
Simulates a click on the object. If the JavaScript event OnClick() or OnChange() are defined for the HTML object, the event will be raised. In most of the cases you do not need to specify a value for the optional parameters.

Parameters:

Part

Description

x

The x position where the click should occur on the current object.

y

The y position where the click should occur on the current object.

eFlag

The flag is only used if the X and Y parameters are defined.
If the flag w3rITEM_CLICK_FLAG_ABSOLUTE is set the click is sent at the absolute X,Y position based on the the web browser control.
If the flag w3rITEM_CLICK_FLAG_CURRENT_POSITION is set the parameters X,Y are set with the current mouse position. See sample below.

Sample:

Page.Controls("buttonOK","Button").Click
Page.Controls("Next Page","Anchor").Click
 
' Click on an image with a specific position
Page.Controls("buyFlights","Image").Click 11,11
 
' Simulate a Left Click on a button
Page.Controls("MyButton","Button").SetFocus
Page.Controls("MyButton","Button").Click ,,w3rITEM_CLICK_FLAG_CURRENT_POSITION+w3rITEM_CLICK_FLAG_WM_MOUSE_DOWN + w3rITEM_CLICK_FLAG_WM_MOUSE_UP

Remark:

By default when the method click is executed from an HTML <INPUT Type='Image' Name='butlogin'> the position of the click is passed in the query string this way as 0,0 :

butLogin.x=0&butLogin.y=0

You can specify a special x and y value if the application requires it, by setting the optional parameters x and y. In this case W3Runner uses the SendMessage Windows Api to simulate a click. The Windows Message WM_LBUTTONDOWN and WM_LBUTTONUP are sent to the Internet Explorer window class "Internet Explorer_Server". X and Y define the position in the image, before sending the message W3Runner converts the values relatively to the Web Browser. If the flag w3rITEM_CLICK_FLAG_ABSOLUTE is set the convertion is not done.

If you set the parameters X and Y and the Image is not visible the click will be ignored. Before the click you can set the focus to the object to force it to be visible.
