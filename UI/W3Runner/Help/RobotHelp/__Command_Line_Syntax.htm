Introduction

It is possible to start a test from the command line. You can also include your own argument and read their values with the property Page.Argument().

Reserved command line arguments name :

Reserved Arguments

Description

-File

The file to load

-Function

The function to execute.

-Left

The left position of the W3Runner main window.

-Top

The top position of the W3Runner main window.

-Width

The width position of the W3Runner main window.

-Height

The height position of the W3Runner main window.

-Fullscreen

The full screen mode.

Syntax :

W3Runner.exe -file "file-name" -function function-name [-argument-name] -[argument-value]

Command Line Sample:

W3Runner.exe -file "c:\mytest\mytest.vbs" -function my_test_function
W3Runner.exe -file "c:\mytest\mytest.vbs" -function my_test_function -MyArgument1 1
W3Runner.exe -file "c:\mytest\mytest.vbs" -function my_test_function -MyArgument2 "Fred"

VBScript Sample:

If Page.Argument("MyArgument1",0)="1" Then
End If
