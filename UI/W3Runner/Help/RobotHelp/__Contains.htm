Contains

Public Function Contains(
ByVal strValue As String,
Optional strContainsCompareMode As String = "Text"
) As Long

Description:
Return the index position of a string in the combox box string list. If the string is not found the return value is -1.

Parameters:

Part

Description

strValue

The value to search.

strContainsCompareMode

Define if the compare is based on the HTML attribute text or value.

Sample:

Page.URL = "http://www.mymail.com"
Page.WaitForDownLoad "login"
 
If Page.Controls("Domain","ComboBox").Contains("mymail.com")<>-1 Then
  Msgbox "Found"
Else
  Msgbox "Not Found"
End If
