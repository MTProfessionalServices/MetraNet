Design Mode

W3Runner combines 3 modes :

  • Design Mode

  • Design Mode + Recod mode on

  • Run Time Mode

The default mode is the Design Mode. In Design mode you can enter the URL in the Address combo box and press enter. The URL will be downloaded and you can click on the different link or button.
Each time a page is downloaded its full URL is logged in the Trace list view.

In Design mode you can click on the Watch Window icon to view all the objects available in the current pages.
