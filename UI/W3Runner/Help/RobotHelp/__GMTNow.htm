GMTNow

Public Function GMTNow(
Optional ByVal strDateSeparator As String = "/",
Optional ByVal strTimeSeparator As String = ":"
) As Variant

Description:

Return the current GMT time in the following format : yyyy/mm/dd hh:mm:ss. The value can be converted into a DATE type with the CDate() function.

Sample:

Dim datNow
datNow = Page.Helper.GMTNow()
