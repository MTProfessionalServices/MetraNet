GetHTMLObjectName

Public Function GetHTMLObjectName(ByVal obj As Object) As String

Description:
Return the name of an object following the method W3Runner uses during the record mode.

Parameters:

Part

Description

obj

A Microsoft Internet Explorer HTML Object (HTMLInputElement, HTMLButtonElement).

Name Look up method:

To find the name of an HTML object W3Runner use :

  • The HTML attribute Name, if not available use

  • The HTML attribute Id, if not available use

  • The HTML attribute Alt, if not available use

  • If the HTML tag is an <HTMLObjectElement> use the HTML attribute Code or ClassID

  • If the HTML tag is an <IMG> use the HTML attribute Src , file name only (no path)

  • If the HTML tag is an <INPUT> use the HTML attribute Value

  • If the HTML tag is an <BUTTON> use the HTML InnerText

  • If the HTML tag is an <A>, <TD> or a <DIV> use the full InnerText of the object. Spaces at the beginning and at the end are ignored.

  • Else return an empty string

Sample:

Dim objIEHTMLObjectTextBox
Set objIEHTMLObjectTextBox = Page.Controls("LastName","TextBox").Object
MsgBox Page.GetHTMLObjectName(objIEHTMLObjectTextBox)
