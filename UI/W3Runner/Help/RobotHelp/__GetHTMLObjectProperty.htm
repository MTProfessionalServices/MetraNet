GetHTMLObjectProperty

Public Function GetHTMLObjectProperty(
ByVal obj As Object,
ByVal strPropertyName As String
) As Variant

Description:
Return an HTML attribute value from an HTML object.

Parameters:

Part

Description

obj

A Microsoft Internet Explorer HTML Object.

strPropertyName

The name of the property

Sample:

Dim objIEHTMLObjectTextBox
Set objIEHTMLObjectTextBox = Page.Controls("LastName","TextBox").Object
MsgBox Page.GetHTMLObjectProperty(objIEHTMLObjectTextBox,"Type")
