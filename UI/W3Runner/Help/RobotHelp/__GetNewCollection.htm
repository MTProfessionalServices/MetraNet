GetNewCollection

Public Function GetNewCollection() As Collection

Description:

Return a new instance of the Visual Basic Collection class. This objects is not available in VBScript by default. For more information see the Visual Basic help. You can also use VBScript dictionary object.

Sample:

Dim objMyCollection, varItem
Set objMyCollection = Page.Helper.GetNewCollection()
 
objMyCollection.Add "A"
objMyCollection.Add "B"
objMyCollection.Add "C"
For Each varItem In objMyCollection
  W3Runner.TRACE varItem
