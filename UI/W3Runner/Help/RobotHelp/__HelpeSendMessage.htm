SendMessage

Public Function SendMessage(ByVal hwnd As Long, ByVal wMsg As Long, ByVal wParam As Long, ByVal lParam As Long) As Long

Description:

Windows API SendMessage(). To send a Windows message to the Web Browser you must use the property Page.WebBrowserHWND. See the Microsoft help for more information.

Sample:

Page.Helper.SendMessage Page.WebBrowserHWND,WM_LBUTTONDOWN,67687
