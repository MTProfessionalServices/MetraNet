Item

Public Property Get Item(ByVal varKey As Variant) As Variant

Description:

This property allows the support of the HTML object <MAP> and the collection of the HTML object <AREA>. You can obtain the same result by the using the Microsoft Internet Explorer Object Model syntax, but this property support look up by name or by index.

Parameters:

Part

Description

varKey

The index or the name of the <AREA> item object to return.

HTML Sample:

<img Name="imgAmerica" src="northamerica.gif" usemap="#Map" border="0">
<map name="Map">
<area Name="Quebec" shape="circle" coords="217, 141, 24" href="map2.asp?state=qc">
<area Name="California" shape="circle" coords="149, 384, 21" href="map2.asp?state=ca">

VBScript Sample:

' W3Runner Standard Syntax
Page.Controls("MyMap","Map").Item("Quebec").Click
Page.WaitForDownLoad ".asp"
Page.Controls("MyMap","Map").Item(0).Click
Page.WaitForDownLoad ".asp"
 
' Microsoft Internet Explorer MSHTML Object Model Syntax
Page.Controls("MyMap","Map").Object.ChildNodes(0).Click
Page.WaitForDownLoad ".asp"
