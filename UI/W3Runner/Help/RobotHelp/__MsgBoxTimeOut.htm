MsgBoxTimeOut

Public Property Get MsgBoxTimeOut() As Long
Public Property Let MsgBoxTimeOut(ByVal v As Long)

Description:
Get or set the MsgBox method time out. The default value can also be set from the Options dialog.

Sample:

W3Runner.MsgBoxTimeOut = 5
W3Runner.MsgBox "Start the Test"
W3Runner.MsgBoxTimeOut = 20
