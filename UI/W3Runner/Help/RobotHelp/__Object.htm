Object

Public Property Get Object() As Object

Description:

Return a reference of the Microsoft Internet Explorer object. For more information see the Microsoft online help.

Sample:

' The type is W3RunnerControl
MsgBox TypeName(Page.Controls("FirstName","TextBox"))
' The type is HTMLInputElement
MsgBox TypeName(Page.Controls("FirstName","TextBox").Object)
 
Page.Controls("MyBody",,"main").Object.doScroll "pagedown"
