W3Runner_OnTrace

PUBLIC SUB W3Runner_OnTrace(ByVal eTraceType, ByVal strMessage)
END SUB

Description:
If the function W3Runner_OnTrace is implemented and set, W3Runner will call this event each time a message is traced .

Parameters:

Part

Description

strMessage

Contains the message that was logged in the log file and displayed in the Trace listview.

Sample:

PUBLIC SUB Main()
 
  W3Runner.TraceEvent = "W3Runner_OnTrace"
  ...
END FUNCTION
 
 ' Log all the error in a special file
PUBLIC SUB W3Runner_OnTrace(ByVal eTraceType, ByVal strMessage)
 
  If eTraceType=w3rERROR Then
 
    W3Runner.TextFile.LogFile W3Runner.Environ("TEMP") & "\W3Runner_OnTrace.log",eTraceType & "," & strMessage
  End If
END SUB
