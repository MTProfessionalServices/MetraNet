Record Mode

You can start the record mode by clicking on the record icon in the tool bar.

  • Recording URL : Enter the URL in the address combo box and press Enter. The VBScript source code which sets and loads the URL is automatically generated.

  • Record User Actions : Press the ALT key and click on the object you want the VBScript source code to be generated

To stop the recording press the stop recording icon. The generated script is then loaded in the current editor. Copy and paste the generated code in you work file.

Remark :

We do not recommend to use the temporary file used by W3Runner as your main file.
