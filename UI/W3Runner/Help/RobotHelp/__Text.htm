Text

Public Property Get Text(Optional strTextType As String = "innerText") As String
Public Property Let Text(ByVal vNewValue As String)

Description:
Gets or sets the text of the object. Supported object : Only supported by TextBox and ComboBox.

Remark:
It not possible to get the text of a password input box.

Parameters:

Part

Description

strTextType

This optional parameter is supported only by the type TabelCell (<TD>), Microsoft Internet Explorer DHTML property to be used :InnerText, OuterText, InnerHTML, OuterHTML..
