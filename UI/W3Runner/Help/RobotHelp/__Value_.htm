Value

Public Property Get Value(Optional ByVal lngIndex As Variant) As Variant
Public Property Let Value(Optional ByVal lngIndex As Variant, ByVal vNewValue As Variant)

Description:
Get or set the value of the object. Only supported by CheckBox, RadioButton, ListBox and ComboBox.

Parameters:

Part

Description

lngIndex

The index of the string to be returned only supported for a ListBox or ComboBox.

Sample:

Page.Controls("Country","ComboBox","main").Value = "ITALY"
Page.Controls("TaxExempt","CheckBox","main").Value = vbChecked
Page.Controls("SubScriberType","RadioButton","main",,"Gold").Value = TRUE
