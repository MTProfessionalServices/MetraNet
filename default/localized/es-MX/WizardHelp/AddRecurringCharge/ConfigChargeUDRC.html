[es-MX]Unit Name
[es-MX]The name appearing next to the unit entry field in MetraCare. For instance, Number of Conferencing Seats.

[es-MX]Min Unit Value and Max Unit Value
[es-MX]The maximum and minimum amounts the user can enter in this field.

[es-MX]Value Type
[es-MX]The permitted value type for this field (either integer or decimal).

[es-MX]Rating
  • [es-MX]Tiered : The system rates all units based on the final range reached in the rate schedule.
  • [es-MX]Tapered : The system rates each unit based on the range the unit falls into in the rate schedule.
