Add Miscellaneous Attribute

MetraOffer Advanced Options Manage AMP Decision Types <selected decision type>

To configure this decision type to perform RAMP decision processing, or to set up calculation of early termination charges, see the Miscellaneous Attributes page.

This feature requires advanced knowledge of AMP processing.

  1. In the Miscellaneous Attributes area, click Add.
  2. Provide the following information.

    Field

    Provide this value

    Name

    The name of the attribute.

    Value
    • Fixed Value—Specify a value.
    • Get from Param Table Column—In the dropdown list, select the column
  3. Click OK.
  4. Continue with the Error Check page.
