Add Log Entry

Account Menu BarRelationships Add Log Entry

Use this page to add an event to an account's audit log.

  1. Choose an Event Type: Note or Telephone Call (if you're entering the event in response to a call).
  2. Type a description of the event.
  3. Click OK.

    The text you entered will appear in the application audit log.
