Use this page to provide general information for a new custom rate.

The Billing Manager in your organization is the person who determines custom rates.

To set a custom rate:

  1. Enter a description for this custom rate.
  2. Enter a start date.
  3. Check Next Billing Period if appropriate. If you check it, the new rate will become effective on the appropriate day and time in the next billing period.
  4. Enter end date. You do not need to enter an end date; it is possible to use this rate indefinitely.
  5. Check Next Billing Period if appropriate. If you check it, the new rate will become inactive on the appropriate day and time in the next billing period.
  6. Click OK. You go to the page where you set the rates.
